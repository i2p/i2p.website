I2P 릴리스 0.6.1.32
=====================

0.6.1.32 릴리스에는 여러 버그 수정, 이전 tunnel 빌드 성공 수준에 따라 tunnel 빌드 강도를 조절하는 zzz의 새로운 tunnel 빌드 알고리즘, 그리고 기타 성능 개선이 포함되어 있습니다.

이 릴리스는 하위 호환되며, 이미 0.6.1.31이 설치되어 있다면 서명된 업데이트 파일의 서명을 인식하므로(자동으로 업데이트할 수 있습니다).

그러나 우리의 서명된 업데이트 서버 3개 간에 부하를 더 고르게 분산하기 위해, 다음의 선택적 확인을 수행하시길 권장합니다:

* Open the update configuration page on:

http://localhost:7657/configupdate.jsp

* If you see 3 update URLs, no action is needed.

업데이트 URL이 하나만 보이면, http://localhost:7657/configadvanced.jsp 에서 고급 구성 페이지를 열고 "router.updateURL="로 시작하는 구성 항목을 삭제하십시오. 그런 다음 "Apply"를 클릭하면 업데이트 URL에 기본 구성이 적용되어야 합니다.
