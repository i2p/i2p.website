I2P 릴리스 0.7
====================

0.7 릴리스는 I2PSnark의 안정성과 유연성을 향상시켜, 향후 I2P 업데이트를 배포하는 데 활용될 수 있기를 기대합니다.

I2P router에는 다양한 전송 계층 및 스트리밍 관련 문제, 네트워크 탐색, NetDB 성능, 그리고 UDP introducer(소개자) 시스템에 대한 수정과 최적화가 이루어졌습니다. 그 외에도 새 릴리스는 더 나은 연결 제한 기능, 헬퍼 애플리케이션에서 발생하는 "out of memory" 예외에 대한 더 높은 내성, 그리고 destination 키(목적지 키)의 Base32 해시를 사용하는 실험적인 새로운 주소 체계(".b32.i2p" URL)을 제공합니다.

BOB 및 SAM 프로토콜이 개선되었고, 더 많은 구형 구성 요소가 제거되었으며, Router Console 기능이 추가되었고, 잠재적인 지연 시간 측정 공격이 완화되었습니다. - 이번 릴리스부터는 문제를 일으키는 피어에 대한 차단 목록이 기본적으로 활성화됩니다.

지난 릴리스부터 I2P는 Java 1.5 이상이 필요하다는 점을 다시 한 번 알려드립니다. 사용 중인 Java 버전이 확실하지 않다면, 터미널 창 또는 명령 프롬프트를 열고 "java -version" 명령을 입력하여 확인할 수 있습니다. 구버전의 Java가 설치되어 있다면 먼저 업데이트해 주세요!

이번 릴리스는 하위 호환됩니다: 이미 0.6.1.31 이상이 설치되어 있다면, 서명된 업데이트 파일의 서명을 인식하여 자동으로 업데이트할 수 있습니다.
