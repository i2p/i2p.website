0.9.14.1에는 i2psnark의 "Add Torrent" 양식과 기타 일부 웹 양식에 대한 수정이 포함되어 있습니다. 콘솔을 통해 플러그인을 설치하는 기능을 다시 사용할 수 있게 했지만, 먼저 router.config 파일(in ~/.i2p/ or /var/lib/i2p/i2p-config/ or %APPDATA%\I2P\)을 편집하여 routerconsole.enablePluginInstall=true 줄을 추가해야 합니다. 0.9.14에서 제거되었던 그 밖의 드물게 사용되는 고급 기능들도 routerconsole.advanced=true 줄을 추가하면 복원할 수 있습니다.

평소와 마찬가지로, 기본값인 "Download and Verify"로 설정되어 있으면, router가 업데이트를 자동으로 다운로드하고 재시작 버튼을 표시합니다. 그러나 0.9.14의 버그로 인해 업데이트가 "Notify only"로 설정되어 있으면 다운로드 버튼이 표시되지 않습니다. 업데이트하려면 설정을 "Download and Verify" 또는 "Download, Verify, and Restart"로 변경해야 합니다.

여전히 0.9.13 또는 그 이전 버전을 사용 중이라면 가능한 한 빨리 이번 릴리스로 업데이트하시길 권장합니다. router console을 자주 확인하지 않으신다면, 항상 최신 릴리스를 실행할 수 있도록 설정을 "Download, Verify, and Restart"로 변경하는 것을 고려해 주세요.

## 릴리스 세부 정보

### Bug Fixes

- Fix i2psnark add torrent form
- Fix iptunnel custom options form
- Fix update download buttons

### 버그 수정

- Restore all console features if routerconsole.advanced=true
- Restore plugin install if routerconsole.enablePluginInstall=true
- Restore client adds/changes if routerconsole.enableClientChange=true
- Plugin signing keys are now whitelisted unless routerconsole.allowUntrustedPlugins=true
- More escaping and cleanups in forms and messages
- Update GeoIP data (new installs and PPA only)
