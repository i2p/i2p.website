2.8.2는 2.8.1 릴리스에서 도입되어 SHA256 실패를 유발하던 버그를 수정합니다. 이 버그는 주로 고대역폭 router에 영향을 미칩니다.

항상 그렇듯이, 이번 릴리스로 업데이트하시기를 권장합니다. 보안을 유지하고 네트워크에 기여하는 가장 좋은 방법은 최신 릴리스를 실행하는 것입니다.

## 릴리스 세부 정보

### Bug Fixes

- Fix SHA256 double-free from Noise
- Clear i2ptunnel bubble count when clearing status messages

### 버그 수정

- Reduce memory usage in BanlistRenderer
- Use torrent name instead of torrent file name in notifications
- Translation updates
