La versión 0.9.14 incluye correcciones críticas para vulnerabilidades de XSS y de ejecución remota reportadas por Exodus Intelligence. Como precaución adicional, hemos deshabilitado varias funciones de configuración avanzadas en la consola del router, incluida la instalación de nuevos complementos. Planeamos volver a habilitarlas en una versión futura tras una revisión adicional.

Debido a cambios en la biblioteca de I2P, los usuarios de I2P-Bote deben actualizar su complemento a la versión 0.2.10 para que funcione con I2P 0.9.14. Su router debería actualizar el complemento automáticamente después de que el router se reinicie.

Esta versión también incluye varias correcciones de errores en i2ptunnel, i2psnark y otras áreas, y actualizaciones a las últimas versiones de Jetty, Tomcat y Wrapper. También hemos implementado un método más rápido y más seguro para el reseeding. Por supuesto, también está la colección habitual de correcciones de errores menores y actualizaciones de traducciones.

Debe actualizar a esta versión de inmediato. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.

## DETALLES DE LA VERSIÓN

### Security Fixes

- Fix several XSS issues
- Disable changing news feed URL from UI
- Disable plugin install
- Disable setting unsigned update URL from UI
- Disable clients.config editing from the UI
- Add Content-Security-Policy and X-XSS-Protection headers
- Disable unused ExecNamingService (thx joernchen of Phenoelit)

### Correcciones de seguridad

- Fix tunnel building so it doesn't get "stuck" on a single pool
- Reject participating tunnels when hidden
- Several i2psnark improvements and fixes (GUI and DHT), including changes for better compatibility with Vuze

### Correcciones de errores

- Reseeding now fetches a signed zip file containing router infos for security and speed
- Use JVM's AES implementation if it is faster
- More advanced options shown in the i2ptunnel edit pages
- Per-message reliabilitiy settings in I2CP and error propagation back from router to client
- Lots of findbugs fixes and cleanups
- Support signature types in SAM, bump rev to 3.1
- New event log page in console
- Jetty 8.1.15.v20140411
- Tomcat 6.0.41
- Wrapper 3.5.25 (new installs and PPA only)
- Translation updates
- Update GeoIP data (new installs and PPA only)
