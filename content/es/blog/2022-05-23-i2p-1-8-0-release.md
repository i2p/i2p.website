## Detalles de la actualización

Esta versión incluye correcciones de errores en i2psnark, el router, I2CP y UPnP. Las correcciones del router abordan errores en el reinicio suave, IPv6, las pruebas entre pares de SSU, las operaciones de almacenamiento de la base de datos de red y la construcción de tunnel. La gestión de las familias de router y la clasificación Sybil también se han mejorado significativamente.

Junto con i2pd, estamos desarrollando nuestro nuevo transporte UDP, SSU2. SSU2 aportará mejoras sustanciales de rendimiento y seguridad. También nos permitirá por fin reemplazar nuestro último uso del muy lento cifrado ElGamal, completando la actualización completa de la criptografía que iniciamos hace unos 9 años. Esta versión incluye una implementación preliminar que está deshabilitada de forma predeterminada. Si desea participar en las pruebas, busque la información actual en zzz.i2p.

Como de costumbre, recomendamos actualizar a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.

## DETALLES DEL LANZAMIENTO

**Cambios**

- SSU2: Preliminary implementation
- Tomcat 9.0.62

**Correcciones de errores**

- Crypto: Fix handling of EdDSA certs with official OID
- I2CP: Fix external connections when session ID is 0
- I2PSnark: Fix size calculation causing tracker errors
- I2PSnark standalone: More fixes and improvements
- Router: Family fixes and improvements
- Router: Fix database store logic
- Router: Fix invalid store and rekeying at startup
- Router: Don't use he.net addresses for GeoIP
- Router: Soft restart fixes
- SSU: Peer test fixes
- SSU: Publish empty IPv6 address when missing introducers
- SSU: Reduce ack delay to improve performance
- Transport: Fix UPnP deadlock
- Tunnels: Don't build client tunnels shorter than minimum length
- UPnP: IPv6 fixes

**Otros**

- API version: 0.9.54
- Console: Setup wizard redesign
- i2psnark: Load system mime types if available
- I2PSnark standalone: Add system tray
- Router: Reduce build reply timeout so we will retry faster
- Router: Avoid Sybil penalty for trusted large families
- Source: Remove BOB
- Translation updates
