2.8.2 corrige un error que causaba fallos de SHA256 y que fue introducido en la versión 2.8.1. El error afecta principalmente a routers de alto ancho de banda.

Como de costumbre, recomendamos actualizar a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.

## Detalles de la versión

### Bug Fixes

- Fix SHA256 double-free from Noise
- Clear i2ptunnel bubble count when clearing status messages

### Correcciones de errores

- Reduce memory usage in BanlistRenderer
- Use torrent name instead of torrent file name in notifications
- Translation updates
