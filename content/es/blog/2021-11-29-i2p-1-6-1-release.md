## Detalles de la actualización

Nota: 1.6.1 es la versión actual. 1.6.0 fue etiquetada pero no publicada.

Esta versión completa el despliegue de dos importantes actualizaciones del protocolo desarrolladas en 2021. Se acelera la transición al cifrado X25519 para routers, y esperamos que casi todos los routers hayan renovado sus claves para finales de año. Se habilitan los mensajes cortos de construcción de tunnel para lograr una reducción significativa del ancho de banda.

Añadimos un panel de selección de tema al nuevo asistente de instalación. Hemos mejorado el rendimiento de SSU y corregido un problema con los mensajes de prueba de pares de SSU. El filtro de Bloom utilizado en la construcción de tunnel se ajustó para reducir el uso de memoria. Hemos mejorado la compatibilidad con plugins no Java.

En otras noticias, estamos haciendo excelentes progresos en el diseño de nuestro nuevo transporte UDP SSU2 y esperamos comenzar la implementación a principios del próximo año.

Como de costumbre, recomendamos actualizar a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.

## DETALLES DE LA VERSIÓN

**Cambios**

- Console: Add theme selection to new-install wizard
- Jetty 9.3.30.v20211001
- Router: Increase probability of rekey to X25519
- SSU: Performance improvements
- SSU: Improve security of peer test
- Tomcat 9.0.54
- Tunnels: Enable sending new short build messages

**Correcciones de errores**

- i2ptunnel: Fix encryption type selection logic

**Otros**

- Improve support for non-Java plugins
- Router: Reduce tunnel build Bloom filter memory usage
- Updates: More changes to support bundled updaters
- Translation updates
