I2P 2.6.1 se publica para corregir un error de la interfaz de usuario en la aplicación Hidden Services Manager. Este error provocaba que el desplazamiento se deshabilitara, dejando parte de la configuración inaccesible.

Como de costumbre, recomendamos que actualices a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.

## Detalles de la versión

### Changes

- Graphs: render graphs in svg
- Translations: generate internal translation leaderboard

### Cambios

- i2ptunnel: Fix bug causing truncation configuration UI

### Correcciones de errores

- Translation updates
