## Versión de corrección de errores para usuarios de Windows 11

Esta versión menor incluye una corrección de error en el router de I2P incluido, que resuelve un error muy poco común en el que el reloj de contexto está desincronizado con el reloj que utiliza el sistema de archivos, lo que da como resultado un router incapaz de leer el estado actual de su propia NetDB. Aunque hasta ahora este error solo se ha observado en Windows 11, se recomienda encarecidamente que todos los usuarios actualicen a la nueva compilación.

This release also features faster startup and times improved stability in the profile manager.
