0.9.11 continúa mejorando la búsqueda y el almacenamiento de LeaseSet para evitar que un atacante correlacione una Destination (destino en I2P) con un Router (enrutador de I2P). Añade compatibilidad con el plugin de outproxy (proxy de salida) [Orchid](http://www.subgraph.com/orchid.html), disponible en [plugins.i2p](http://plugins.i2p/plugins/orchid/). Se reduce el uso de memoria gracias a correcciones en los transportes. Contamos con algunas mejoras del protocolo I2CP que proporcionarán mejores capacidades de búsqueda y protección mediante autorización para clientes externos. Por supuesto, también está la habitual colección de correcciones de errores. Todos los usuarios deberían actualizar.

Esta podría ser la última versión que funciona con Java 5, el cual es muy antiguo y ya no tiene soporte. Si está usando un entorno de ejecución de Java 5 o 6, recomendamos encarecidamente que actualice a Java 7.

Como de costumbre, recomendamos que actualice a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.

## DETALLES DE LA VERSIÓN

### Anonymity Improvements

- More leaseset handling improvements

### Mejoras en el anonimato

- Fix NPE after client shutdown
- Fix wrapper log encoding on logs page
- Streaming ping and I2Ping fixes

### Correcciones de errores

- Add support for Orchid plugin
- Add HTTPS support to HTTP client proxy
- New I2CP support for hostname lookups by external clients
- Stricter I2CP authorization enforcement of external clients (incompatible change)
- Increase default inbound exploratory tunnel length variance
- Big reduction in memory usage by transports
- All in-net updates via torrent
- Translation updates
- Update GeoIP data (new installs and PPA only)
