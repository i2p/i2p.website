0.9.13 incluye correcciones para routers detrás de un cortafuegos, mejoras en las consultas de netDb y una gran actualización de SusiMail. Por supuesto, también hay la habitual colección de correcciones de errores y actualizaciones de traducción.

Como de costumbre, recomendamos actualizar a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.

zzz ha actualizado sus claves GPG, y los archivos del lanzamiento están firmados con sus nuevas claves. La huella digital de su nueva clave es: 2D3D 2D03 910C 6504 C121 0C65 EE60 C0C8 EE72 56A8

## DETALLES DE LA VERSIÓN

### SusiMail

- Many UI improvements
- Implement local storage of messages
- Add offline mode
- Messages now deleted on server after download
- Several backend POP3 and SMTP speedups and fixes

### SusiMail

- NetDB lookup fixes
- Fix transition from not-firewalled to firewalled
- Fix plugin uninstall on Windows
- SSU locking fixes
- Fix rapid republishing of SSU addresses
- IRC client exception fixes
- Fix changing HTTP outproxy configuration without restarting tunnel

### Correcciones de errores

- New i2ptunnel server option for unique local address per-client
- Warn in i2ptunnel on duplicate client ports
- Update HTTP User-Agent to match TBB
- Extend SSU establishment retransmission timer
- Use constant-time method for HMAC verification
- New translation: Slovak
- Translation updates
- Update GeoIP data (new installs and PPA only)
