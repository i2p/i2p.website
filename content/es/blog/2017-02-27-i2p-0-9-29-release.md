## Detalles de la actualización

La versión 0.9.29 incluye correcciones para numerosos tickets de Trac, incluidas soluciones alternativas para mensajes comprimidos corrompidos. Ahora admitimos NTP sobre IPv6. Hemos añadido compatibilidad preliminar con Docker. Ahora contamos con man pages (páginas del manual) traducidas. Ahora reenviamos encabezados Referer del mismo origen a través del proxy HTTP. Hay más correcciones para Java 9, aunque todavía no recomendamos Java 9 para uso general.

Como de costumbre, recomendamos actualizar a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.
