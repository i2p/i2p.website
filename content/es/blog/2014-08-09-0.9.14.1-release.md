0.9.14.1 incluye correcciones para el formulario "Añadir torrent" en i2psnark y algunos otros formularios web. Hemos restaurado la posibilidad de instalar complementos a través de la consola, pero primero debe editar su archivo router.config (en ~/.i2p/ o /var/lib/i2p/i2p-config/ o %APPDATA%\I2P\) para añadir la línea routerconsole.enablePluginInstall=true. Otras funciones avanzadas poco utilizadas que se eliminaron en 0.9.14 pueden restaurarse añadiendo la línea routerconsole.advanced=true.

Como de costumbre, si está configurado con el valor predeterminado "Download and Verify", el router descargará la actualización automáticamente y mostrará un botón para reiniciar. Sin embargo, debido a un error en 0.9.14, si la actualización está configurada para "Notify only", no se mostrará el botón de descarga. Debe cambiar su configuración a "Download and Verify" o "Download, Verify, and Restart" para actualizar.

Si aún está ejecutando la versión 0.9.13 o una anterior, le recomendamos actualizar a esta versión lo antes posible. Si no consulta con frecuencia la consola del router, por favor considere cambiar su configuración a "Download, Verify, and Restart" para asegurarse de que siempre esté ejecutando la versión más reciente.

## DETALLES DE LA VERSIÓN

### Bug Fixes

- Fix i2psnark add torrent form
- Fix iptunnel custom options form
- Fix update download buttons

### Correcciones de errores

- Restore all console features if routerconsole.advanced=true
- Restore plugin install if routerconsole.enablePluginInstall=true
- Restore client adds/changes if routerconsole.enableClientChange=true
- Plugin signing keys are now whitelisted unless routerconsole.allowUntrustedPlugins=true
- More escaping and cleanups in forms and messages
- Update GeoIP data (new installs and PPA only)
