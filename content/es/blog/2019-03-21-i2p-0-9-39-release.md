## Detalles de la actualización

La 0.9.39 incluye cambios extensos para los nuevos tipos de base de datos de red (propuesta 123). El plugin i2pcontrol ahora se distribuye como una aplicación web para respaldar el desarrollo de aplicaciones RPC. Se han corregido varios errores importantes en streaming, i2psnark y NTCP2.

Como de costumbre, recomendamos actualizar a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.
