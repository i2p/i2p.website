Después de la publicación de I2P 2.2.0, que se adelantó para acelerar las mitigaciones frente a los ataques DDOS, detectamos algunos problemas emergentes que hicieron necesario compilar y publicar nuevos paquetes. Esta versión corrige un problema en Ubuntu Lunar y Debian Sid en el que la consola del router era inaccesible al usar una versión actualizada del paquete jakarta. Los paquetes de Docker no leían correctamente los argumentos, lo que daba como resultado archivos de configuración inaccesibles. Este problema también se ha resuelto. El contenedor de docker ahora también es compatible con Podman.

Esta versión sincroniza las traducciones con Transifex y actualiza la base de datos de GeoIP.

Como de costumbre, recomendamos actualizar a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.

## Detalles


### Changes

- Fix missing Java options in docker/rootfs/startapp.sh
- Detect when running in Podman instead of regular Docker
- Update Tor Browser User-Agent String
- Update local GeoIP database
- Remove invalid signing keys from old installs
- Update Tomcat version in Ubuntu Lunar and Debian Sid
