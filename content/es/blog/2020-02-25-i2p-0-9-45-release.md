## Detalles de la actualización

0.9.45 contiene correcciones importantes para el modo oculto y el probador de ancho de banda. Hay una actualización del tema oscuro de la consola. Seguimos trabajando en mejorar el rendimiento y en el desarrollo de un nuevo cifrado de extremo a extremo (propuesta 144).

Como de costumbre, recomendamos actualizar a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la última versión.

## DETALLES DE LA VERSIÓN

**Cambios**

- Dark theme improvements
- Console icon changes
- Move some translations from console to router and core for embedded uses
- Jetty 9.2.29
- Tomcat 8.5.50
- i2psnark HTML5 preview player for partial files

**Correcciones de errores**

- Numerous bandwidth test fixes
- Hidden mode fixes to prevent losing peers
- Hebrew translation fixes

**Otros**

- Content Security Policy improvements
- SSU extend timeout for large messages
- ECIES-X25519 continued development and fixes
- Cancel timeout jobs on message reply
- Add SSL and SSL redirect support to EepGet
- Translation updates
- Update JBigI library
