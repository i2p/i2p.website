# Propuestas de Mejora de I2P

Esta sección contiene todas las Propuestas de Mejora de I2P (IEP) - documentos formales que describen cambios, mejoras y avances para la red I2P y sus especificaciones.

## Sobre el Proceso de Propuestas

Las propuestas de I2P siguen un proceso estructurado adaptado del proceso de propuestas de Tor. Cada propuesta pasa por varias etapas:

- **Abierta**: Bajo discusión
- **Aceptada**: Lista para implementar
- **Terminada**: Aceptada e implementada
- **Cerrada**: Integrada en las especificaciones principales
- **Rechazada**: No se implementará

## Encontrar Propuestas

Utiliza la tabla a continuación para navegar por todas las propuestas. Puedes filtrar por estado, buscar por título o autor, y ver información detallada de cada propuesta.

**Nota**: Las propuestas 000-099 están reservadas para propuestas meta y especiales. Las propuestas de características comienzan desde 100.

Para más información sobre el proceso de propuestas en sí, consulta [Propuesta 001 - El Proceso de Propuestas de I2P](/proposals/001).
