## Update details

0.9.38 enables NTCP2 by default for all users. NTCP2 provides effective resistance against DPI censorship and results in reduced CPU load because of the faster, modern cryptography used. The release also includes important bug fixes and performance improvements.

As usual, we recommend that you update to this release. The best way to maintain security and help the network is to run the latest release.
