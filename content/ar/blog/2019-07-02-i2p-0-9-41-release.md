## Update details

0.9.41 includes performance improvements, streaming library enhancements, and important bug fixes. We continue development work on new leasesets (LS2) and encrypted leasesets. The release contains the usual assortment of bug fixes and translation updates.

As usual, we recommend that you update to this release. The best way to maintain security and help the network is to run the latest release.
