2.8.2 يصلح خللاً يؤدي إلى فشل SHA256 كان قد أُدخل في إصدار 2.8.1. يؤثر هذا الخلل بشكل أساسي على routers ذات عرض نطاق عالٍ.

كالعادة، نوصي بأن تقوم بالتحديث إلى هذا الإصدار. أفضل طريقة للحفاظ على الأمان ومساعدة الشبكة هي تشغيل أحدث إصدار.

## تفاصيل الإصدار

### Bug Fixes

- Fix SHA256 double-free from Noise
- Clear i2ptunnel bubble count when clearing status messages

### إصلاحات الأخطاء

- Reduce memory usage in BanlistRenderer
- Use torrent name instead of torrent file name in notifications
- Translation updates
