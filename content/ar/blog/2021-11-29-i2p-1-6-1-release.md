## تفاصيل التحديث

ملاحظة: 1.6.1 هو الإصدار الحالي. تم وسم 1.6.0 ولكن لم يتم إصداره.

يستكمل هذا الإصدار عملية طرح تحديثين كبيرين للبروتوكول تم تطويرهما في عام 2021. يتم تسريع الانتقال إلى تشفير X25519 للـ routers، ونتوقع إعادة توليد المفاتيح لمعظم الـ routers بحلول نهاية العام. تم تفعيل رسائل إنشاء الـ tunnel القصيرة لتقليل استهلاك النطاق الترددي بشكل ملحوظ.

أضفنا لوحة لاختيار السمة إلى معالج التثبيت الجديد. حسّنا أداء SSU وأصلحنا مشكلة في رسائل اختبار الأقران الخاصة بـ SSU. تم تعديل مرشح بلوم الخاص ببناء الـ tunnel لتقليل استخدام الذاكرة. عزّزنا دعم المكوّنات الإضافية غير المكتوبة بلغة Java.

وفي خبر آخر، نحرز تقدمًا ممتازًا في تصميم آلية النقل الجديدة عبر UDP المسماة SSU2 ونتوقع بدء التنفيذ في وقت مبكر من العام القادم.

كالعادة، نوصي بالتحديث إلى هذا الإصدار. أفضل طريقة للحفاظ على الأمان ومساعدة الشبكة هي تشغيل أحدث إصدار.

## تفاصيل الإصدار

**التغييرات**

- Console: Add theme selection to new-install wizard
- Jetty 9.3.30.v20211001
- Router: Increase probability of rekey to X25519
- SSU: Performance improvements
- SSU: Improve security of peer test
- Tomcat 9.0.54
- Tunnels: Enable sending new short build messages

**إصلاحات الأخطاء**

- i2ptunnel: Fix encryption type selection logic

**أخرى**

- Improve support for non-Java plugins
- Router: Reduce tunnel build Bloom filter memory usage
- Updates: More changes to support bundled updaters
- Translation updates
