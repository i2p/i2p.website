## Update details

0.9.34 includes important bug fixes for hidden services, streaming, and SAM. The release continues work on preparation for Ed25519 as the default signature type and NTCP2 as a new transport. Performance improvements and the usual assortment of bug fixes are included.

As usual, we recommend that you update to this release. The best way to maintain security and help the network is to run the latest release.
