2.8.2 corrige un bogue entraînant des échecs de SHA256 qui a été introduit dans la version 2.8.1. Ce bogue affecte principalement les routers à bande passante élevée.

Comme d'habitude, nous vous recommandons de mettre à jour vers cette version. La meilleure façon de maintenir la sécurité et d'aider le réseau est d'exécuter la dernière version.

## Détails de la version

### Bug Fixes

- Fix SHA256 double-free from Noise
- Clear i2ptunnel bubble count when clearing status messages

### Corrections de bogues

- Reduce memory usage in BanlistRenderer
- Use torrent name instead of torrent file name in notifications
- Translation updates
