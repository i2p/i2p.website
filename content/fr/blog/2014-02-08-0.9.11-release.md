0.9.11 continue d’améliorer la recherche et le stockage des LeaseSet afin d’empêcher un attaquant de corréler une Destination avec un Router. Il ajoute la prise en charge du plugin outproxy (proxy de sortie) [Orchid](http://www.subgraph.com/orchid.html) qui est disponible sur [plugins.i2p](http://plugins.i2p/plugins/orchid/). On observe une réduction de la consommation mémoire grâce à des correctifs dans les transports. Nous avons apporté quelques améliorations au protocole I2CP qui fourniront de meilleures capacités de recherche et une protection d’autorisation pour les clients externes. Bien sûr, il y a aussi la collection habituelle de corrections de bogues. Tous les utilisateurs devraient mettre à jour.

Il se peut que ce soit la dernière version à fonctionner avec Java 5, qui est très ancien et n’est plus pris en charge. Si vous utilisez un environnement d’exécution Java 5 ou 6, nous vous recommandons vivement de passer à Java 7.

Comme d'habitude, nous vous recommandons de mettre à jour vers cette version. La meilleure façon de préserver la sécurité et d'aider le réseau est d'utiliser la dernière version.

## DÉTAILS DE LA VERSION

### Anonymity Improvements

- More leaseset handling improvements

### Améliorations de l'anonymat

- Fix NPE after client shutdown
- Fix wrapper log encoding on logs page
- Streaming ping and I2Ping fixes

### Corrections de bogues

- Add support for Orchid plugin
- Add HTTPS support to HTTP client proxy
- New I2CP support for hostname lookups by external clients
- Stricter I2CP authorization enforcement of external clients (incompatible change)
- Increase default inbound exploratory tunnel length variance
- Big reduction in memory usage by transports
- All in-net updates via torrent
- Translation updates
- Update GeoIP data (new installs and PPA only)
