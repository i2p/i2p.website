## Détails de la mise à jour

La 0.9.45 contient des correctifs importants pour le mode caché et l’outil de test de bande passante. Le thème sombre de la console a été mis à jour. Nous poursuivons le travail visant à améliorer les performances et à développer un nouveau chiffrement de bout en bout (proposition 144).

Comme d'habitude, nous vous recommandons de mettre à jour vers cette version. La meilleure façon de maintenir la sécurité et d'aider le réseau est d'exécuter la dernière version.

## DÉTAILS DE LA VERSION

**Changements**

- Dark theme improvements
- Console icon changes
- Move some translations from console to router and core for embedded uses
- Jetty 9.2.29
- Tomcat 8.5.50
- i2psnark HTML5 preview player for partial files

**Corrections de bogues**

- Numerous bandwidth test fixes
- Hidden mode fixes to prevent losing peers
- Hebrew translation fixes

**Autres**

- Content Security Policy improvements
- SSU extend timeout for large messages
- ECIES-X25519 continued development and fixes
- Cancel timeout jobs on message reply
- Add SSL and SSL redirect support to EepGet
- Translation updates
- Update JBigI library
