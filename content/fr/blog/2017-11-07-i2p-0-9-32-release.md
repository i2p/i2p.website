## Détails de la mise à jour

0.9.32 contient un certain nombre de correctifs dans la console du router et les applications web associées (addressbook, i2psnark et susimail). Nous avons également modifié la façon dont nous gérons les noms d’hôte configurés pour les router infos publiées, afin d’éliminer certaines attaques d’énumération du réseau via le DNS. Nous avons ajouté des vérifications dans la console pour résister aux attaques de rebinding.

Comme d'habitude, nous vous recommandons de mettre à jour vers cette version. La meilleure façon de maintenir la sécurité et d'aider le réseau est d'utiliser la dernière version.
