La version 0.9.13 inclut des correctifs pour les routers derrière un pare-feu, des améliorations de la recherche netdb, et une importante mise à jour de SusiMail. Bien sûr, il y a aussi la collection habituelle de corrections de bogues et de mises à jour des traductions.

Comme d'habitude, nous vous recommandons de mettre à jour vers cette version. La meilleure façon de maintenir la sécurité et d'aider le réseau est d'utiliser la dernière version.

zzz a mis à jour ses clés GPG, et les fichiers de la version sont signés avec ses nouvelles clés. L’empreinte de sa nouvelle clé est : 2D3D 2D03 910C 6504 C121 0C65 EE60 C0C8 EE72 56A8

## DÉTAILS DE LA VERSION

### SusiMail

- Many UI improvements
- Implement local storage of messages
- Add offline mode
- Messages now deleted on server after download
- Several backend POP3 and SMTP speedups and fixes

### SusiMail

- NetDB lookup fixes
- Fix transition from not-firewalled to firewalled
- Fix plugin uninstall on Windows
- SSU locking fixes
- Fix rapid republishing of SSU addresses
- IRC client exception fixes
- Fix changing HTTP outproxy configuration without restarting tunnel

### Corrections de bogues

- New i2ptunnel server option for unique local address per-client
- Warn in i2ptunnel on duplicate client ports
- Update HTTP User-Agent to match TBB
- Extend SSU establishment retransmission timer
- Use constant-time method for HMAC verification
- New translation: Slovak
- Translation updates
- Update GeoIP data (new installs and PPA only)
