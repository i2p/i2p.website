0.9.14.1 inclut des correctifs pour le formulaire "Add Torrent" dans i2psnark et pour quelques autres formulaires Web. Nous avons rétabli la possibilité d'installer des plugins via la console, mais vous devez d'abord modifier votre fichier router.config (dans ~/.i2p/ ou /var/lib/i2p/i2p-config/ ou %APPDATA%\I2P\) pour ajouter la ligne routerconsole.enablePluginInstall=true. D'autres fonctionnalités avancées rarement utilisées, qui avaient été supprimées dans la 0.9.14, peuvent être rétablies en ajoutant la ligne routerconsole.advanced=true.

Comme d'habitude, si le router est configuré avec le paramètre par défaut "Download and Verify", il téléchargera automatiquement la mise à jour et affichera un bouton pour redémarrer. Cependant, en raison d'un bogue dans la version 0.9.14, si votre mise à jour est configurée sur "Notify only", le bouton de téléchargement ne sera pas affiché. Vous devez modifier votre configuration en "Download and Verify" ou "Download, Verify, and Restart" pour effectuer la mise à jour.

Si vous exécutez encore la 0.9.13 ou une version antérieure, nous vous recommandons de mettre à jour vers cette version dès que possible. Si vous ne consultez pas souvent votre console du router, veuillez envisager de modifier votre configuration sur "Download, Verify, and Restart" afin de vous assurer que vous exécutez toujours la dernière version.

## DÉTAILS DE LA VERSION

### Bug Fixes

- Fix i2psnark add torrent form
- Fix iptunnel custom options form
- Fix update download buttons

### Corrections de bogues

- Restore all console features if routerconsole.advanced=true
- Restore plugin install if routerconsole.enablePluginInstall=true
- Restore client adds/changes if routerconsole.enableClientChange=true
- Plugin signing keys are now whitelisted unless routerconsole.allowUntrustedPlugins=true
- More escaping and cleanups in forms and messages
- Update GeoIP data (new installs and PPA only)
