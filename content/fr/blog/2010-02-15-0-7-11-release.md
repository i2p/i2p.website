La version 0.7.11 corrige plusieurs problèmes, dont certains liés aux horloges système, qui empêchaient les nouveaux routers de s’intégrer rapidement au réseau. Elle corrige également le programme d’installation pour Windows 64 bits. Cette version inclut une mise à jour vers [Jetty 5.1.15](http://old.nabble.com/jetty-5.1.15,-6.1.18-and-7.0.0.M2-td23652819.html) pour corriger [cette vulnérabilité](http://www.kb.cert.org/vuls/id/402580).

Comme d’habitude, cette version contient davantage d’améliorations en matière d’anonymat et de performances, ainsi qu’une série de correctifs de bogues. La mise à jour est recommandée.

Veuillez faire part de vos retours aux développeurs sur IRC #i2p ou sur [forum.i2p2.de](http://forum.i2p2.de/), et [impliquez-vous](http://www.i2p2.de/getinvolved.html), faites passer le mot et [faites un don](http://www.i2p2.de/donate.html) ! Nous sommes toujours à la recherche d’aide pour les traductions nouvelles et existantes. Veuillez vous porter volontaire sur IRC #i2p.

Les fichiers sont disponibles sur la [page de téléchargement](/downloads/).

Détails de la version :

## Programme d'installation

- Fix installations on 64-bit Windows by upgrading to launch4j 3.0.1 and adding 64-bit dll to izpack.
- The wrapper is still 32-bit only. 64-bit Windows users should start I2P with the 'No Window' icon if the 'Restartable' icon does not work.

## Transport

- Better handle clock skews at startup, even if NTP does not work because UDP is blocked or other reasons
- Fix clock skew reporting
- Bootstrap profiles when loading the netDb from disk (broken in 0.7.8)

## Serveur web intégré (Jetty)

- Upgrade to [Jetty 5.1.15](http://old.nabble.com/jetty-5.1.15,-6.1.18-and-7.0.0.M2-td23652819.html) which fixes [this vulnerability](http://www.kb.cert.org/vuls/id/402580).
- Turn on checkAliases for increased protection

## Base de données réseau

- Add some DOS protection for floodfills
- Increase the number of floodfills again

## Autres

- Faster startup
- Bug fixes in i2psnark, BOB, clock handling, and more
