Cette version non planifiée désactive les messages de vérification RouterInfo qui ont été utilisés dans l’attaque publiée dans l’article de l’UCSB, ce qui devrait rendre la corrélation entre un LeaseSet et un Router beaucoup plus difficile. Nous avons également inclus un nombre limité d’autres correctifs énumérés ci-dessous. Notre version 0.9.8, qui inclura la prise en charge d’IPv6, est toujours dans les temps pour fin septembre.

Comme d'habitude, nous recommandons à tous les utilisateurs de mettre à jour vers cette version.

## DÉTAILS DE LA VERSION

### Anonymity Improvements

- Disable RouterInfo verification messages

### Améliorations de l'anonymat

- Extend inbound tunnel expiration
- i2prouter: bashism fix
- i2psnark: increase max piece size, mime type updates
- New reseed host
- New update hosts, thanks Meeh and dg
- Streaming: RTO changes
- Updater: Increase update-via-torrent to 30 percent
- UPnP fix for some hardware
