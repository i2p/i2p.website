## Détails de la mise à jour

Le programme d’installation Windows 0.9.29, publié sur notre site web il y a deux jours, échoue à l’installation en levant une exception. Nous avons corrigé le bogue et mis en ligne le nouveau programme d’installation sur notre site web. Ceci n’affecte pas les mises à niveau depuis les versions précédentes ni les programmes d’installation non Windows. Nous vous présentons nos excuses pour ce problème.
