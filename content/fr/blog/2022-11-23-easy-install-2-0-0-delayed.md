## Détails de la mise à jour

Le bundle Easy-Install pour Windows a été publié. Dans cette version, la prise en charge a été ajoutée pour la plupart des principaux navigateurs, y compris tous les principaux dérivés de Firefox(Gecko) et de Chromium. La compatibilité avec les installations externes du service I2P et les installations I2P hors bundle effectuées par les utilisateurs a été améliorée. Le bundle Easy-Install peut désormais détecter d'autres I2P routers et inviter l'utilisateur à les lancer à la place, si l'utilisateur dispose déjà d'I2P. Les extensions de navigateur ont été mises à jour vers les dernières versions. Le bundle Easy-Install a désormais accès au mode de convivialité de `i2p.plugins.firefox` via l'option de ligne de commande `-usability`. Le mode par défaut est le mode "Strict" où le Javascript est désactivé par NoScript. En mode de convivialité, le Javascript est restreint par JShelter. Pour plus de détails, voir le dépôt du gestionnaire de profils sur i2pgit.org.

Il est recommandé de mettre à jour vers cette version pour une sécurité, une confidentialité et des performances optimales, ainsi que pour aider le réseau.

En raison du départ du développeur/mainteneur, l’Easy-Install Bundle pour OSX sera retardé d’un mois, le temps d’organiser la reprise du rôle de mainteneur.
