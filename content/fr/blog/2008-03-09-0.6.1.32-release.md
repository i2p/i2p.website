I2P version 0.6.1.32 =====================

La version 0.6.1.32 contient plusieurs corrections de bogues, un nouvel algorithme de construction de tunnels de zzz qui régule l’intensité de la construction de tunnels en fonction des taux de réussite antérieurs, et d’autres améliorations des performances.

Cette version est rétrocompatible et, si vous avez déjà la 0.6.1.31 installée, elle reconnaîtra la signature du fichier de mise à jour signé (vous pourrez ainsi effectuer la mise à jour automatiquement).

Toutefois, afin d’assurer une meilleure répartition de la charge entre nos 3 serveurs de mise à jour signés, je vous recommande d’effectuer la vérification optionnelle suivante :

* Open the update configuration page on:

http://localhost:7657/configupdate.jsp

* If you see 3 update URLs, no action is needed.

Si vous ne voyez qu’une seule URL de mise à jour, ouvrez la page de configuration avancée sur http://localhost:7657/configadvanced.jsp et supprimez l’entrée de configuration commençant par "router.updateURL=". Cliquez ensuite sur "Apply", et la configuration par défaut devrait prendre effet pour les URL de mise à jour.

* Please note that since 0.6.1.31, configuration entries

peut s’étendre sur plusieurs lignes ; veuillez donc vérifier attentivement où commence l’entrée suivante, afin de ne pas l’endommager !
