La version 0.8.4 contient des améliorations de performances et des correctifs de bogues importants. De plus, i2psnark prend désormais en charge les liens magnet. Comme d'habitude, il est recommandé de mettre à jour.

Merci d’aider à développer le réseau. [Participez](http://www.i2p2.de/getinvolved.html), parlez-en autour de vous, et [faites un don](http://www.i2p2.de/donate.html) ! Si vous trouvez un bug, veuillez soumettre un rapport sur [gitlab](http://i2pgit.org/i2p-hackers/i2p.i2p/issues). Nous recherchons toujours de l’aide pour les nouvelles traductions et celles déjà existantes. Merci de vous porter volontaire sur IRC #i2p-dev.

Les fichiers sont disponibles sur la [page de téléchargement](/downloads/).

## Détails de la version :

- Fix a severe memory leak in router I2CP session management that caused router crashes for people running the Robert bittorrent client
- Fix a bug from 0.8.2 that filtered cookies in the HTTP Server tunnel, causing authentication problems for some eepsites (I2P Sites)
- Several fixes for rare NPEs

## I2PSnark

- Magnet and "maggot" link support
- Peer Exchange
- Metadata transfer
- Details page enhancements
- Bandwidth limiter improvements
- Bug fixes

## Autres

- More performance improvements
- Adjustment to peer selection, to hopefully use more of the available network capacity
- Translation updates
- Start of a new Arabic translation by 'hamada'
