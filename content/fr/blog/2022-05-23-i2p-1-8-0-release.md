## Détails de la mise à jour

Cette version inclut des correctifs de bogues dans i2psnark, le router, I2CP et l'UPnP. Les correctifs du router corrigent des bogues liés au redémarrage à chaud, à l'IPv6, aux tests de pairs SSU, aux opérations de stockage dans la base de données réseau et à la construction des tunnels. La gestion des familles de router et la classification Sybil ont également été nettement améliorées.

En collaboration avec i2pd, nous développons notre nouveau transport UDP, SSU2. SSU2 apportera des améliorations substantielles en matière de performances et de sécurité. Il nous permettra également de remplacer enfin notre ultime utilisation du chiffrement ElGamal, très lent, achevant la mise à niveau complète de la cryptographie que nous avons commencée il y a environ 9 ans. Cette version contient une implémentation préliminaire désactivée par défaut. Si vous souhaitez participer aux tests, veuillez consulter les informations à jour sur zzz.i2p.

Comme d’habitude, nous vous recommandons de mettre à jour vers cette version. La meilleure façon de préserver la sécurité et d’aider le réseau est d’utiliser la dernière version.

## DÉTAILS DE LA VERSION

**Modifications**

- SSU2: Preliminary implementation
- Tomcat 9.0.62

**Corrections de bogues**

- Crypto: Fix handling of EdDSA certs with official OID
- I2CP: Fix external connections when session ID is 0
- I2PSnark: Fix size calculation causing tracker errors
- I2PSnark standalone: More fixes and improvements
- Router: Family fixes and improvements
- Router: Fix database store logic
- Router: Fix invalid store and rekeying at startup
- Router: Don't use he.net addresses for GeoIP
- Router: Soft restart fixes
- SSU: Peer test fixes
- SSU: Publish empty IPv6 address when missing introducers
- SSU: Reduce ack delay to improve performance
- Transport: Fix UPnP deadlock
- Tunnels: Don't build client tunnels shorter than minimum length
- UPnP: IPv6 fixes

**Autres**

- API version: 0.9.54
- Console: Setup wizard redesign
- i2psnark: Load system mime types if available
- I2PSnark standalone: Add system tray
- Router: Reduce build reply timeout so we will retry faster
- Router: Avoid Sybil penalty for trusted large families
- Source: Remove BOB
- Translation updates
