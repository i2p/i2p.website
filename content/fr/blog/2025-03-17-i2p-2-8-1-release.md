2.8.1 corrige l'accès aux sites locaux qui ne fonctionnait plus dans la 2.8.0. Nous avons ajouté des bulles de notification à plusieurs applications dans la console afin de mettre en évidence les messages des applications. Nous avons corrigé un bogue dans le test de tunnel qui pouvait affecter la fiabilité des tunnels. Le carnet d'adresses dispose désormais d'un nouvel onglet "latest" qui affiche les noms d'hôte les plus récents provenant de vos abonnements. Il y a plusieurs correctifs et améliorations du thème sombre.

Nous avons corrigé le programme d'installation qui échouait auparavant avec Java 21 ou une version ultérieure. Nous avons également corrigé l'installation dans des chemins de répertoires contenant des espaces sous Linux. Pour ceux d'entre vous qui utilisent Docker, nous avons mis à jour le conteneur Docker et les bibliothèques du conteneur qui étaient obsolètes.

Comme d'habitude, nous vous recommandons de mettre à jour vers cette version. La meilleure façon de préserver la sécurité et d'aider le réseau est d'exécuter la dernière version.

## Détails de la version

### Changes

- New console notification bubbles
- New addressbook sort-by-latest tab
- Add support for .i2p.alt hostnames (RFC 9476)

### Modifications

- Fix local site access bug
- Fix installer failure on Java 21+ (IzPack 5.2.4)
- Fix tunnel tests for outbound client tunnels
- Fix installing to paths with spaces (non-Windows)
- Console dark theme fixes and improvements
- Fix AIOOBE on large signed streaming packets
- Fix compression of Router Identities
- Update outdated Docker container and container libraries

### Corrections de bogues

- Add logo to first installer panel
- Add progress bar to installer panels
- Use SHA256 pool in Noise
- Move certs page to a debug page tab
- Reduce memory usage in AddressBean
- Set bulk profile for servers and i2psnark
- Prep for tunnel bandwidth parameters (proposal 168)
- Proxy error page improvements
- Check key order when parsing RI mappings
- Reduce i2psnark peer check interval
- Translation updates

[Liste complète des bogues corrigés](http://git.idk.i2p/i2p-hackers/i2p.i2p/-/issues?scope=all&state=closed&milestone_title=2.9.0)
