0.8.4 रिलीज़ में कुछ प्रदर्शन सुधार और महत्वपूर्ण त्रुटि सुधार शामिल हैं। साथ ही, i2psnark अब magnet links (मैग्नेट लिंक) का समर्थन करता है। हमेशा की तरह, अपग्रेड करने की सलाह दी जाती है।

कृपया नेटवर्क के विस्तार में मदद करें। [शामिल हों](http://www.i2p2.de/getinvolved.html), इसके बारे में जानकारी फैलाएँ, और [दान करें](http://www.i2p2.de/donate.html)! यदि आपको कोई बग मिले, तो कृपया [gitlab](http://i2pgit.org/i2p-hackers/i2p.i2p/issues) पर एक रिपोर्ट दर्ज करें। हम नए और मौजूदा अनुवादों में मदद के लिए अभी भी स्वयंसेवकों की तलाश कर रहे हैं। कृपया IRC #i2p-dev पर स्वयंसेवा करें।

फ़ाइलें [डाउनलोड पृष्ठ](/downloads/) पर उपलब्ध हैं।

## रिलीज़ विवरण:

- Fix a severe memory leak in router I2CP session management that caused router crashes for people running the Robert bittorrent client
- Fix a bug from 0.8.2 that filtered cookies in the HTTP Server tunnel, causing authentication problems for some eepsites (I2P Sites)
- Several fixes for rare NPEs

## I2PSnark

- Magnet and "maggot" link support
- Peer Exchange
- Metadata transfer
- Details page enhancements
- Bandwidth limiter improvements
- Bug fixes

## अन्य

- More performance improvements
- Adjustment to peer selection, to hopefully use more of the available network capacity
- Translation updates
- Start of a new Arabic translation by 'hamada'
