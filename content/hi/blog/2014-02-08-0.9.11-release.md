0.9.11, किसी हमलावर द्वारा Destination को किसी Router से सहसंबंधित करने से रोकने के लिए LeaseSet लुकअप और स्टोरेज में सुधार जारी रखता है। यह [Orchid](http://www.subgraph.com/orchid.html) outproxy प्लगइन के लिए समर्थन जोड़ता है, जो [plugins.i2p](http://plugins.i2p/plugins/orchid/) पर उपलब्ध है। transports में किए गए सुधारों के कारण मेमोरी उपयोग में कमी आई है। हमने कुछ I2CP प्रोटोकॉल सुधार किए हैं जो बाहरी क्लाइंट्स के लिए बेहतर लुकअप सुविधाएँ और प्राधिकरण सुरक्षा प्रदान करेंगे। स्वाभाविक रूप से, सामान्य बग फिक्स का एक संग्रह भी शामिल है। सभी उपयोगकर्ताओं को अपडेट करना चाहिए।

हो सकता है कि यह Java 5 के साथ काम करने वाला अंतिम रिलीज़ हो, जो बहुत पुराना है और अब समर्थित नहीं है। यदि आप Java 5 या 6 रनटाइम का उपयोग कर रहे हैं, तो हम दृढ़ता से अनुशंसा करते हैं कि आप Java 7 में अपग्रेड करें।

हमेशा की तरह, हम अनुशंसा करते हैं कि आप इस रिलीज़ पर अपडेट करें। सुरक्षा बनाए रखने और नेटवर्क की सहायता करने का सबसे अच्छा तरीका नवीनतम रिलीज़ चलाना है।

## रिलीज़ विवरण

### Anonymity Improvements

- More leaseset handling improvements

### अनामता में सुधार

- Fix NPE after client shutdown
- Fix wrapper log encoding on logs page
- Streaming ping and I2Ping fixes

### त्रुटि सुधार

- Add support for Orchid plugin
- Add HTTPS support to HTTP client proxy
- New I2CP support for hostname lookups by external clients
- Stricter I2CP authorization enforcement of external clients (incompatible change)
- Increase default inbound exploratory tunnel length variance
- Big reduction in memory usage by transports
- All in-net updates via torrent
- Translation updates
- Update GeoIP data (new installs and PPA only)
