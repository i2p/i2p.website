0.7.11 रिलीज़ कई समस्याओं को ठीक करती है, जिनमें से कुछ क्लॉक्स से संबंधित थीं, जिनकी वजह से नए routers नेटवर्क में तेजी से एकीकृत नहीं हो पा रहे थे। यह 64-bit Windows के लिए इंस्टॉलर को भी ठीक करता है। इस रिलीज़ में [Jetty 5.1.15](http://old.nabble.com/jetty-5.1.15,-6.1.18-and-7.0.0.M2-td23652819.html) के लिए एक अपडेट शामिल है ताकि [इस भेद्यता](http://www.kb.cert.org/vuls/id/402580) को ठीक किया जा सके।

हमेशा की तरह, इस रिलीज़ में गुमनामी और प्रदर्शन में और सुधार, तथा विभिन्न बग फिक्स शामिल हैं। अपग्रेड करना अनुशंसित है।

कृपया डेवलपर्स को IRC #i2p पर या [forum.i2p2.de](http://forum.i2p2.de/) पर प्रतिक्रिया दें, [शामिल हों](http://www.i2p2.de/getinvolved.html), इसके बारे में लोगों तक बात पहुँचाएँ, और [दान करें](http://www.i2p2.de/donate.html)! हम नई और मौजूदा अनुवादों के लिए अभी भी मदद की तलाश में हैं। कृपया IRC #i2p पर स्वयंसेवक बनें।

फ़ाइलें [डाउनलोड पृष्ठ](/downloads/) पर उपलब्ध हैं।

रिलीज़ विवरण:

## इंस्टॉलर

- Fix installations on 64-bit Windows by upgrading to launch4j 3.0.1 and adding 64-bit dll to izpack.
- The wrapper is still 32-bit only. 64-bit Windows users should start I2P with the 'No Window' icon if the 'Restartable' icon does not work.

## ट्रांसपोर्ट

- Better handle clock skews at startup, even if NTP does not work because UDP is blocked or other reasons
- Fix clock skew reporting
- Bootstrap profiles when loading the netDb from disk (broken in 0.7.8)

## एंबेडेड वेब सर्वर (Jetty)

- Upgrade to [Jetty 5.1.15](http://old.nabble.com/jetty-5.1.15,-6.1.18-and-7.0.0.M2-td23652819.html) which fixes [this vulnerability](http://www.kb.cert.org/vuls/id/402580).
- Turn on checkAliases for increased protection

## नेटवर्क डेटाबेस

- Add some DOS protection for floodfills
- Increase the number of floodfills again

## अन्य

- Faster startup
- Bug fixes in i2psnark, BOB, clock handling, and more
