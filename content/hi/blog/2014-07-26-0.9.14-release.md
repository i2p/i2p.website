0.9.14 में Exodus Intelligence द्वारा रिपोर्ट की गई XSS और दूरस्थ निष्पादन कमजोरियों के लिए गंभीर सुधार शामिल हैं। अतिरिक्त सावधानी के रूप में, हमने router console में कई उन्नत विन्यास सुविधाएँ, जिनमें नए प्लगइन्स की स्थापना भी शामिल है, अक्षम कर दी हैं। अतिरिक्त समीक्षा के बाद, हम इन्हें किसी आगामी रिलीज़ में पुनः सक्षम करने की योजना बना रहे हैं।

I2P लाइब्रेरी में हुए परिवर्तनों के कारण, I2P-Bote उपयोगकर्ताओं को I2P 0.9.14 के साथ काम करने के लिए अपने प्लगइन को संस्करण 0.2.10 में अपग्रेड करना आवश्यक है। आपका router पुनःआरंभ होने के बाद प्लगइन को स्वचालित रूप से अपडेट कर देना चाहिए।

इस रिलीज़ में i2ptunnel, i2psnark और अन्य क्षेत्रों में कई बग फिक्स शामिल हैं, और Jetty, Tomcat तथा Wrapper को नवीनतम संस्करणों तक अपडेट किया गया है। हमने reseeding (नेटवर्क से प्रारंभिक router जानकारी डाउनलोड करने की प्रक्रिया) के लिए एक तेज़ और अधिक सुरक्षित विधि भी लागू की है। बेशक, हमेशा की तरह, छोटे-मोटे बग फिक्स और अनुवाद अपडेट का सामान्य संग्रह भी शामिल है।

आपको तुरंत इस रिलीज़ पर अपडेट करना होगा। सुरक्षा बनाए रखने और नेटवर्क की मदद करने का सबसे अच्छा तरीका नवीनतम रिलीज़ चलाना है।

## रिलीज़ विवरण

### Security Fixes

- Fix several XSS issues
- Disable changing news feed URL from UI
- Disable plugin install
- Disable setting unsigned update URL from UI
- Disable clients.config editing from the UI
- Add Content-Security-Policy and X-XSS-Protection headers
- Disable unused ExecNamingService (thx joernchen of Phenoelit)

### सुरक्षा सुधार

- Fix tunnel building so it doesn't get "stuck" on a single pool
- Reject participating tunnels when hidden
- Several i2psnark improvements and fixes (GUI and DHT), including changes for better compatibility with Vuze

### दोष सुधार

- Reseeding now fetches a signed zip file containing router infos for security and speed
- Use JVM's AES implementation if it is faster
- More advanced options shown in the i2ptunnel edit pages
- Per-message reliabilitiy settings in I2CP and error propagation back from router to client
- Lots of findbugs fixes and cleanups
- Support signature types in SAM, bump rev to 3.1
- New event log page in console
- Jetty 8.1.15.v20140411
- Tomcat 6.0.41
- Wrapper 3.5.25 (new installs and PPA only)
- Translation updates
- Update GeoIP data (new installs and PPA only)
