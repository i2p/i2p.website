## अद्यतन विवरण

इस रिलीज़ में i2psnark, Router, I2CP और UPnP में बग फ़िक्स शामिल हैं। Router में किए गए फ़िक्स सॉफ्ट रीस्टार्ट, IPv6, SSU पीयर परीक्षण, नेटवर्क डेटाबेस स्टोर्स, और tunnel निर्माण से संबंधित बगों को संबोधित करते हैं। Router family के प्रबंधन और Sybil वर्गीकरण में भी उल्लेखनीय सुधार किए गए हैं।

i2pd के साथ मिलकर, हम अपना नया UDP ट्रांसपोर्ट, SSU2, विकसित कर रहे हैं। SSU2 प्रदर्शन और सुरक्षा में महत्वपूर्ण सुधार लेकर आएगा। यह हमें अत्यंत धीमी ElGamal encryption के हमारे अंतिम उपयोग को अंततः बदलने की अनुमति भी देगा, जिससे लगभग 9 वर्ष पहले शुरू किया गया हमारा पूर्ण क्रिप्टोग्राफी उन्नयन पूरा हो जाएगा। इस रिलीज़ में एक प्रारंभिक कार्यान्वयन शामिल है, जो डिफ़ॉल्ट रूप से निष्क्रिय है। यदि आप परीक्षण में भाग लेना चाहते हैं, तो कृपया zzz.i2p पर वर्तमान जानकारी देखें।

हमेशा की तरह, हम अनुशंसा करते हैं कि आप इस रिलीज़ में अपडेट करें। सुरक्षा बनाए रखने और नेटवर्क की मदद करने का सबसे अच्छा तरीका नवीनतम रिलीज़ चलाना है।

## रिलीज़ विवरण

**परिवर्तन**

- SSU2: Preliminary implementation
- Tomcat 9.0.62

**त्रुटि सुधार**

- Crypto: Fix handling of EdDSA certs with official OID
- I2CP: Fix external connections when session ID is 0
- I2PSnark: Fix size calculation causing tracker errors
- I2PSnark standalone: More fixes and improvements
- Router: Family fixes and improvements
- Router: Fix database store logic
- Router: Fix invalid store and rekeying at startup
- Router: Don't use he.net addresses for GeoIP
- Router: Soft restart fixes
- SSU: Peer test fixes
- SSU: Publish empty IPv6 address when missing introducers
- SSU: Reduce ack delay to improve performance
- Transport: Fix UPnP deadlock
- Tunnels: Don't build client tunnels shorter than minimum length
- UPnP: IPv6 fixes

**अन्य**

- API version: 0.9.54
- Console: Setup wizard redesign
- i2psnark: Load system mime types if available
- I2PSnark standalone: Add system tray
- Router: Reduce build reply timeout so we will retry faster
- Router: Avoid Sybil penalty for trusted large families
- Source: Remove BOB
- Translation updates
