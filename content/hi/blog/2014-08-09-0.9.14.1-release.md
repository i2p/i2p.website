0.9.14.1 में i2psnark के "Add Torrent" फ़ॉर्म और कुछ अन्य वेब फ़ॉर्म्स के लिए सुधार शामिल हैं। हमने कंसोल के माध्यम से प्लगइन्स इंस्टॉल करने की सुविधा बहाल कर दी है, लेकिन इसके लिए पहले आपको अपनी router.config फाइल (~/.i2p/ या /var/lib/i2p/i2p-config/ या %APPDATA%\I2P\ में) संपादित करके यह पंक्ति जोड़नी होगी: routerconsole.enablePluginInstall=true. 0.9.14 में हटाई गई अन्य कम-प्रयोग वाली उन्नत सुविधाएँ भी इस पंक्ति को जोड़कर फिर से उपलब्ध कराई जा सकती हैं: routerconsole.advanced=true.

हमेशा की तरह, यदि डिफ़ॉल्ट "Download and Verify" के साथ कॉन्फ़िगर किया गया है, तो router स्वचालित रूप से अपडेट डाउनलोड करेगा और रीस्टार्ट करने के लिए एक बटन दिखाएगा। हालाँकि, 0.9.14 में एक बग के कारण, यदि आपका अपडेट "Notify only" पर कॉन्फ़िगर है, तो डाउनलोड बटन प्रदर्शित नहीं होगा। अपडेट करने के लिए आपको अपनी कॉन्फ़िगरेशन को "Download and Verify" या "Download, Verify, and Restart" में बदलना होगा।

यदि आप अभी भी 0.9.13 या उससे पुराना संस्करण चला रहे हैं, तो हम सलाह देते हैं कि आप यथाशीघ्र इस रिलीज़ पर अपडेट करें। यदि आप अक्सर अपने router console की जाँच नहीं करते हैं, तो कृपया अपने विन्यास को "Download, Verify, and Restart" में बदलने पर विचार करें ताकि यह सुनिश्चित हो कि आप हमेशा नवीनतम रिलीज़ चला रहे हों।

## रिलीज़ विवरण

### Bug Fixes

- Fix i2psnark add torrent form
- Fix iptunnel custom options form
- Fix update download buttons

### त्रुटि सुधार

- Restore all console features if routerconsole.advanced=true
- Restore plugin install if routerconsole.enablePluginInstall=true
- Restore client adds/changes if routerconsole.enableClientChange=true
- Plugin signing keys are now whitelisted unless routerconsole.allowUntrustedPlugins=true
- More escaping and cleanups in forms and messages
- Update GeoIP data (new installs and PPA only)
