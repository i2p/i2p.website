## Aktualisierungsdetails

0.9.45 enthält wichtige Fehlerbehebungen für den versteckten Modus und den Bandbreitentester. Es gibt ein Update für das dunkle Konsolenthema. Wir arbeiten weiterhin an Leistungsverbesserungen und an der Entwicklung neuer Ende-zu-Ende-Verschlüsselung (Proposal 144).

Wie üblich empfehlen wir, auf diese Version zu aktualisieren. Der beste Weg, die Sicherheit aufrechtzuerhalten und dem Netzwerk zu helfen, ist, die neueste Version zu verwenden.

## DETAILS ZUR VERÖFFENTLICHUNG

**Änderungen**

- Dark theme improvements
- Console icon changes
- Move some translations from console to router and core for embedded uses
- Jetty 9.2.29
- Tomcat 8.5.50
- i2psnark HTML5 preview player for partial files

**Fehlerbehebungen**

- Numerous bandwidth test fixes
- Hidden mode fixes to prevent losing peers
- Hebrew translation fixes

**Sonstiges**

- Content Security Policy improvements
- SSU extend timeout for large messages
- ECIES-X25519 continued development and fixes
- Cancel timeout jobs on message reply
- Add SSL and SSL redirect support to EepGet
- Translation updates
- Update JBigI library

[Vollständige Liste der behobenen Fehler](http://trac.i2p2.i2p/query?resolution=fixed&milestone=0.9.45)
