0.9.14.1 enthält Fehlerbehebungen für das Formular "Add Torrent" in i2psnark und einige andere Webformulare. Wir haben die Möglichkeit, Plugins über die Konsole zu installieren, wiederhergestellt, aber Sie müssen zuvor Ihre Datei router.config (in ~/.i2p/ oder /var/lib/i2p/i2p-config/ oder %APPDATA%\I2P\) bearbeiten, um die Zeile routerconsole.enablePluginInstall=true hinzuzufügen. Andere, selten verwendete erweiterte Funktionen, die in 0.9.14 entfernt wurden, können durch Hinzufügen der Zeile routerconsole.advanced=true wiederhergestellt werden.

Wie üblich lädt der router, wenn er mit der Standardeinstellung "Download and Verify" konfiguriert ist, das Update automatisch herunter und zeigt eine Schaltfläche zum Neustart an. Aufgrund eines Fehlers in 0.9.14 wird jedoch, wenn Ihr Update auf "Notify only" konfiguriert ist, die Download-Schaltfläche nicht angezeigt. Sie müssen Ihre Konfiguration auf "Download and Verify" oder "Download, Verify, and Restart" ändern, um zu aktualisieren.

Wenn Sie noch 0.9.13 oder älter ausführen, empfehlen wir, so bald wie möglich auf diese Version zu aktualisieren. Wenn Sie Ihre router-Konsole nicht häufig überprüfen, erwägen Sie bitte, Ihre Konfiguration auf "Herunterladen, Überprüfen und Neu starten" zu ändern, um sicherzustellen, dass Sie stets die neueste Version ausführen.

## DETAILS ZUR VERÖFFENTLICHUNG

### Bug Fixes

- Fix i2psnark add torrent form
- Fix iptunnel custom options form
- Fix update download buttons

### Fehlerbehebungen

- Restore all console features if routerconsole.advanced=true
- Restore plugin install if routerconsole.enablePluginInstall=true
- Restore client adds/changes if routerconsole.enableClientChange=true
- Plugin signing keys are now whitelisted unless routerconsole.allowUntrustedPlugins=true
- More escaping and cleanups in forms and messages
- Update GeoIP data (new installs and PPA only)
