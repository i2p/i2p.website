I2P-Release 0.6.5
====================

Das Release 0.6.5 führt neue Komponenten ein, entfernt einige alte (wie den alten TCP-Transport) und wurde optimiert, um eine bessere Leistung zu erzielen.

Das BOB (Basic Open Bridge)-Protokoll wird eingeführt, zur Verwendung durch Client-Anwendungen, die I2CP-Bibliotheken nicht direkt importieren können. Damit wird das alte SAM-Protokoll, das bisher in solchen Fällen verwendet wurde, als veraltet eingestuft. Vorläufig wird BOB bei Neuinstallationen jedoch noch nicht automatisch gestartet, und auf bestehenden Installationen bleibt SAM aktiv.

Verbesserter Code sollte besser in der Lage sein, Überlastung zu verhindern, indem er probabilistisch mitvermittelten Datenverkehr verwirft, und sich ebenso besser verhalten, wenn Überlastung auftritt. Die floodfill NetDB sollte zuverlässiger arbeiten, die Streaming-Bibliothek sollte bessere Nachrichtengrößen wählen, eine Socket-Timeout-Funktion bieten, und die Arbeiten am "hidden"-Betriebsmodus für I2P routers schreiten voran.

- From this release onward, I2P requires Java 1.5 or higher.

Wenn Sie sich über Ihre Java-Version nicht sicher sind, können Sie dies überprüfen, indem Sie ein Terminalfenster oder eine Eingabeaufforderung öffnen und den Befehl "java -version" eingeben. Wenn Sie eine ältere Java-Version installiert haben, aktualisieren Sie diese bitte zuerst!

Diese Version ist abwärtskompatibel: Wenn Sie bereits 0.6.1.31 oder höher installiert haben, wird sie die Signatur der signierten Update-Datei erkennen, und Sie können automatisch aktualisieren.
