I2P-Version 0.7
====================

Die Version 0.7 bringt I2PSnark mehr Stabilität und Flexibilität, sodass es hoffentlich künftig zur Verteilung von I2P-Updates genutzt werden kann.

Der I2P router erhält Fehlerbehebungen und Optimierungen für verschiedene Probleme auf der Transportebene und beim Streaming, bei der Netzwerkerkundung, der NetDB-Leistung und dem UDP-Introducer-System. Unter anderem bietet die neue Version eine bessere Begrenzung von Verbindungen, eine höhere Toleranz gegenüber "out of memory"-Ausnahmen in Hilfsanwendungen und ein experimentelles neues Adresssystem, das Base32-Hashes von Destination-Schlüsseln verwendet (".b32.i2p" URLs).

Sowohl die BOB- als auch die SAM-Protokolle wurden verbessert, weitere veraltete Komponenten entfernt, Funktionen der Router Console hinzugefügt und ein möglicher Latenzmessangriff entschärft. - Ab dieser Version sind Blocklisten für Peers mit Fehlverhalten standardmäßig aktiviert.

Es scheint sinnvoll, daran zu erinnern, dass I2P bereits seit der letzten Veröffentlichung Java 1.5 oder höher benötigt. Wenn Sie sich über Ihre Java-Version nicht sicher sind, können Sie dies überprüfen, indem Sie ein Terminalfenster oder eine Eingabeaufforderung öffnen und den Befehl "java -version" eingeben. Wenn Sie eine ältere Java-Version installiert haben, aktualisieren Sie diese bitte zuerst!

Diese Version ist abwärtskompatibel: Wenn Sie bereits 0.6.1.31 oder höher installiert haben, erkennt sie die Signatur der signierten Update-Datei und Sie können automatisch aktualisieren.
