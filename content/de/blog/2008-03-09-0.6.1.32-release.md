I2P-Version 0.6.1.32 =====================

Das Release 0.6.1.32 enthält mehrere Fehlerbehebungen, einen neuen Algorithmus für den tunnel-Aufbau von zzz, der die Intensität des tunnel-Aufbaus entsprechend den bisherigen Erfolgsraten reguliert, sowie weitere Leistungsverbesserungen.

Diese Version ist abwärtskompatibel, und wenn Sie bereits 0.6.1.31 installiert haben, wird sie die Signatur der signierten Update-Datei erkennen (damit Sie automatisch aktualisieren können).

Um jedoch eine bessere Lastverteilung zwischen unseren 3 signierten Update-Servern sicherzustellen, empfehle ich, dass Sie die folgende optionale Überprüfung durchführen:

* Open the update configuration page on:

http://localhost:7657/configupdate.jsp

* If you see 3 update URLs, no action is needed.

Wenn Sie nur eine Update-URL sehen, öffnen Sie die erweiterte Konfigurationsseite unter http://localhost:7657/configadvanced.jsp und löschen Sie den Konfigurationseintrag, der mit "router.updateURL=" beginnt. Klicken Sie dann auf "Apply", und die Standardkonfiguration sollte für die Update-URLs gelten.

* Please note that since 0.6.1.31, configuration entries

kann sich über mehrere Zeilen erstrecken, daher bitte sorgfältig prüfen, wo der nächste Eintrag beginnt, um Beschädigungen zu vermeiden!
