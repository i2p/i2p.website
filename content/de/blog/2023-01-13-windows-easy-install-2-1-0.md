## Aktualisierungsdetails

Das I2P Easy-Install-Bundle für Windows in Version 2.1.0 wurde veröffentlicht. Wie üblich enthält dieses Release eine aktualisierte Version vom I2P Router. Dieses Release von I2P bietet verbesserte Strategien zum Umgang mit Netzwerküberlastung. Diese sollten Leistung und Konnektivität verbessern und die langfristige Gesundheit des I2P-Netzwerks sichern.

Diese Version bringt überwiegend Verbesserungen unter der Haube am Starter für Browser-Profile. Die Kompatibilität mit dem Tor Browser Bundle wurde verbessert, indem die TBB-Konfiguration über Umgebungsvariablen aktiviert wurde. Das Firefox-Profil wurde aktualisiert, und die Basisversionen der Erweiterungen wurden aktualisiert. In der gesamten Codebasis und im Bereitstellungsprozess wurden Verbesserungen vorgenommen.

Leider ist dieses Release noch ein unsigniertes .exe-Installationsprogramm. Bitte überprüfen Sie die Prüfsumme des Installationsprogramms, bevor Sie es verwenden. Die Updates hingegen sind mit meinen I2P-Signierschlüsseln signiert und daher sicher.

Dieses Release wurde mit OpenJDK 19 kompiliert. Es verwendet i2p.plugins.firefox in Version 1.0.7 als Bibliothek zum Starten des Browsers. Es verwendet i2p.i2p in Version 2.1.0 als I2P router und zur Bereitstellung von Anwendungen. Wie immer wird empfohlen, den I2P router so bald wie möglich auf die neueste Version zu aktualisieren.
