## 更新详细信息

0.9.32 在 router 控制台及相关的 web 应用（addressbook、i2psnark 和 susimail）中包含若干修复。我们还更改了对已发布的 router infos 的已配置主机名的处理方式，以消除某些通过 DNS 进行的网络枚举攻击。我们在 router 控制台中增加了一些检查，以抵御 DNS 重绑定攻击。

一如既往，我们建议您更新到此版本。维护安全并帮助网络的最佳方式是运行最新版本。
