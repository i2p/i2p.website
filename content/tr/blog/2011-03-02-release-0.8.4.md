0.8.4 sürümü bazı performans iyileştirmeleri ve önemli hata düzeltmeleri içerir. Ayrıca, i2psnark artık magnet bağlantılarını destekliyor. Her zamanki gibi, güncelleme yapmanız önerilir.

Please help grow the network. [Get involved](http://www.i2p2.de/getinvolved.html), spread the word, and [donate](http://www.i2p2.de/donate.html)! If you find a bug, please enter a report on [gitlab](http://i2pgit.org/i2p-hackers/i2p.i2p/issues). We are still looking for help on new and existing translations. Please volunteer on IRC #i2p-dev.

Dosyalar [indirme sayfasında](/downloads/) mevcuttur.

## Sürüm ayrıntıları:

- Fix a severe memory leak in router I2CP session management that caused router crashes for people running the Robert bittorrent client
- Fix a bug from 0.8.2 that filtered cookies in the HTTP Server tunnel, causing authentication problems for some eepsites (I2P Sites)
- Several fixes for rare NPEs

## I2PSnark

- Magnet and "maggot" link support
- Peer Exchange
- Metadata transfer
- Details page enhancements
- Bandwidth limiter improvements
- Bug fixes

## Diğer

- More performance improvements
- Adjustment to peer selection, to hopefully use more of the available network capacity
- Translation updates
- Start of a new Arabic translation by 'hamada'
