I2P 2.2.0 sürümü, DDOS saldırılarına karşı önlemleri hızlandırmak amacıyla öne alındıktan sonra, yeni paketlerin oluşturulup yayımlanmasını gerekli kılan bazı yeni ortaya çıkan sorunlar tespit edildi. Bu sürüm, Ubuntu Lunar ve Debian Sid üzerinde güncellenmiş jakarta paketi kullanıldığında router console’a erişilememesi sorununu düzeltiyor. Docker paketleri argümanları doğru okumadığından, yapılandırma dosyalarına erişilememesine yol açıyordu. Bu sorun da giderildi. Docker kapsayıcısı artık Podman ile de uyumlu.

Bu sürüm çevirileri transifex ile senkronize eder ve GeoIP veritabanını günceller.

Her zamanki gibi, bu sürüme güncellemenizi öneririz. Güvenliği korumanın ve ağa yardımcı olmanın en iyi yolu en son sürümü çalıştırmaktır.

## Ayrıntılar

### Changes

- Fix missing Java options in docker/rootfs/startapp.sh
- Detect when running in Podman instead of regular Docker
- Update Tor Browser User-Agent String
- Update local GeoIP database
- Remove invalid signing keys from old installs
- Update Tomcat version in Ubuntu Lunar and Debian Sid
