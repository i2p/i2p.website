2.8.1, 2.8.0'da bozulan yerel sitelere erişim sorunlarını gideriyor. Uygulama iletilerini vurgulamak için konsoldaki çeşitli uygulamalara bildirim baloncukları ekledik. Tunnel testine ilişkin, tunnel güvenilirliğini etkiliyor olabilecek bir hatayı giderdik. Adres defteri artık aboneliklerinizden eklenen en yeni ana bilgisayar adlarını görüntüleyen yeni bir "latest" sekmesine sahip. Koyu tema ile ilgili çeşitli düzeltmeler ve iyileştirmeler var.

Java 21 ve üzeri sürümlerde daha önce başarısız olan yükleyiciyi düzelttik. Ayrıca Linux'ta adı boşluk içeren dizin yollarına kurulumla ilgili sorunu da düzelttik. Docker kullananlar için, güncel olmayan Docker konteynerlerini ve konteyner kütüphanelerini güncelledik.

Her zamanki gibi, bu sürüme güncellemenizi öneriyoruz. Güvenliği korumanın ve ağa yardımcı olmanın en iyi yolu en son sürümü çalıştırmaktır.

## Sürüm Ayrıntıları

### Changes

- New console notification bubbles
- New addressbook sort-by-latest tab
- Add support for .i2p.alt hostnames (RFC 9476)

### Değişiklikler

- Fix local site access bug
- Fix installer failure on Java 21+ (IzPack 5.2.4)
- Fix tunnel tests for outbound client tunnels
- Fix installing to paths with spaces (non-Windows)
- Console dark theme fixes and improvements
- Fix AIOOBE on large signed streaming packets
- Fix compression of Router Identities
- Update outdated Docker container and container libraries

### Hata Düzeltmeleri

- Add logo to first installer panel
- Add progress bar to installer panels
- Use SHA256 pool in Noise
- Move certs page to a debug page tab
- Reduce memory usage in AddressBean
- Set bulk profile for servers and i2psnark
- Prep for tunnel bandwidth parameters (proposal 168)
- Proxy error page improvements
- Check key order when parsing RI mappings
- Reduce i2psnark peer check interval
- Translation updates

[Giderilen hataların tam listesi](http://git.idk.i2p/i2p-hackers/i2p.i2p/-/issues?scope=all&state=closed&milestone_title=2.9.0)
