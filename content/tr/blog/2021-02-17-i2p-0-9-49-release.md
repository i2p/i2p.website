## Güncelleme ayrıntıları

0.9.49, I2P'yi daha hızlı ve daha güvenli hale getirme çalışmalarını sürdürüyor. Daha yüksek hızlar sağlaması beklenen SSU (UDP) taşıma protokolü için çeşitli iyileştirmeler ve düzeltmelerimiz var. Bu sürüm ayrıca router'lar için yeni, daha hızlı ECIES-X25519 şifrelemeye geçişi de başlatıyor. (Destinations (hedef kimlikler) bu şifrelemeyi birkaç sürümdür zaten kullanıyor) Birkaç yıldır yeni şifreleme için spesifikasyonlar ve protokoller üzerinde çalışıyoruz ve artık işin sonuna yaklaşıyoruz! Geçişin tamamlanması birkaç sürüm sürecek.

Bu sürümde, aksaklıkları en aza indirmek için, yalnızca yeni kurulumlar ve mevcut kurulumların çok küçük bir yüzdesi (yeniden başlatma sırasında rastgele seçilen) yeni şifrelemeyi kullanacaktır. Router’ınız yeni şifrelemeyi kullanmak için "rekey" (anahtarları yenilemek) gerçekleştirirse, yeniden başlatmadan sonraki birkaç gün boyunca router’ınızın trafiği daha düşük olabilir veya güvenilirliği azalabilir. Bu normaldir, çünkü router’ınız yeni bir kimlik oluşturmuştur. Performansınız bir süre sonra eski haline dönmelidir.

Daha önce varsayılan imza türünü değiştirirken ağı iki kez "rekey" (anahtarların yenilenmesi) yaptık, ancak varsayılan şifreleme türünü ilk kez değiştiriyoruz. Umarız her şey sorunsuz ilerler, ancak emin olmak için yavaş başlıyoruz.

Her zamanki gibi, bu sürüme güncellemenizi öneririz. Güvenliği korumanın ve ağa yardımcı olmanın en iyi yolu, en son sürümü çalıştırmaktır.

## SÜRÜM AYRINTILARI

**Değişiklikler**

- Build: Git migration
- Build: Move web resources to wars
- i2psnark WebSeed support
- i2psnark padding file support
- i2ptunnel: Move proxy resources to jar
- Router: Redesign ECIES encryption for floodfills (proposal 156)
- Router: Verify RI stores after startup
- Router: Reduce Sybil threshold
- Router: ECIES for new routers
- Router: Start of ECIES migration
- SSU: Send individual fragments of messages
- SSU: Westwood+ congestion control
- SSU: Fast retransmit

**Hata Düzeltmeleri**

- Build: Fix Gradle build
- Crypto: Increase ratchet tag window to prevent message loss
- I2CP: Fix encrypted leaseset combined with ECIES crypto or offline keys
- i2ptunnel: Fix config file saving issues
- Router: Fix leaseset request fails causing watchdog to bark
- Router: Hidden mode fixes
- SSU: Fix partial acks not being sent
- SSU: Fix occasional high CPU usage

**Diğer**

- Crypto: AES performance improvements
- DoH: Change to RFC 8484 style
- i2ptunnel: Remove DSA shared clients
- Proxy: Add jump servers
- Router: Add more countries for hidden mode
- Router: Tunnel peer selection changes
- Router: Move Sybil subsystem from console to router for embedded use
- Router: Verify RI stores for a while after startup
- Util: New unit tests
- Translation updates
