## Güncelleme ayrıntıları

Bu sürüm, i2psnark, router, I2CP ve UPnP'deki hata düzeltmelerini içerir. Router düzeltmeleri, soft restart, IPv6, SSU eş testi, ağ veritabanına kaydetme ve tunnel oluşturma ile ilgili hataları giderir. Router family yönetimi ve Sybil sınıflandırması da önemli ölçüde iyileştirildi.

i2pd ile birlikte, yeni UDP transport (taşıma protokolü)müz SSU2'yi geliştiriyoruz. SSU2, önemli performans ve güvenlik iyileştirmeleri getirecek. Ayrıca, yaklaşık 9 yıl önce başlattığımız kriptografi yükseltmesini tamamlayarak, çok yavaş ElGamal şifrelemesinin kalan son kullanımını nihayet değiştirmemize olanak tanıyacak. Bu sürüm, varsayılan olarak devre dışı bırakılmış ön bir uygulama içeriyor. Testlere katılmak isterseniz, lütfen güncel bilgiler için zzz.i2p adresine bakın.

Her zamanki gibi, bu sürüme güncellemenizi öneririz. Güvenliği korumanın ve ağa yardımcı olmanın en iyi yolu en son sürümü çalıştırmaktır.

## SÜRÜM AYRINTILARI

**Değişiklikler**

- SSU2: Preliminary implementation
- Tomcat 9.0.62

**Hata Düzeltmeleri**

- Crypto: Fix handling of EdDSA certs with official OID
- I2CP: Fix external connections when session ID is 0
- I2PSnark: Fix size calculation causing tracker errors
- I2PSnark standalone: More fixes and improvements
- Router: Family fixes and improvements
- Router: Fix database store logic
- Router: Fix invalid store and rekeying at startup
- Router: Don't use he.net addresses for GeoIP
- Router: Soft restart fixes
- SSU: Peer test fixes
- SSU: Publish empty IPv6 address when missing introducers
- SSU: Reduce ack delay to improve performance
- Transport: Fix UPnP deadlock
- Tunnels: Don't build client tunnels shorter than minimum length
- UPnP: IPv6 fixes

**Diğer**

- API version: 0.9.54
- Console: Setup wizard redesign
- i2psnark: Load system mime types if available
- I2PSnark standalone: Add system tray
- Router: Reduce build reply timeout so we will retry faster
- Router: Avoid Sybil penalty for trusted large families
- Source: Remove BOB
- Translation updates
