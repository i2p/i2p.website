## Güncelleme ayrıntıları

0.9.44, gizli servislerin yeni şifreleme türlerini ele alma sürecindeki bir hizmet reddi (DoS) sorununa yönelik önemli bir düzeltme içeriyor. Tüm kullanıcılar mümkün olan en kısa sürede güncelleme yapmalıdır.

Sürüm, yeni uçtan uca şifreleme (Öneri 144) için başlangıç düzeyinde destek içeriyor. Bu projedeki çalışmalar sürüyor ve henüz kullanıma hazır değil. Konsol ana sayfasında değişiklikler ve i2psnark içinde yeni gömülü HTML5 medya oynatıcıları var. Güvenlik duvarı arkasındaki IPv6 ağları için ek düzeltmeler de dahil edildi. Tunnel oluşturma ile ilgili düzeltmeler, bazı kullanıcılar için daha hızlı başlatma sağlamalıdır.

Her zamanki gibi, bu sürüme güncellemenizi öneriyoruz. Güvenliği korumanın ve ağa yardımcı olmanın en iyi yolu en son sürümü çalıştırmaktır.

## Sürüm Ayrıntıları

**Değişiklikler**

- Console home page changes
- Initial code for new encryption (proposal 144)
- Router changes to support multiple encryption types
- HTML5 players in i2psnark
- Audio playlist in i2psnark
- Change Windows data location to %LOCALAPPDIR% (new installs only)

**Hata Düzeltmeleri**

- Fix AppArmor config for Java 11
- Fix handling of unsupported encryption types
- IPv6 firewalled fixes
- Fix long stop delay for some plugins
- Fix custom icons for some plugins
- Fix exploratory tunnel build issues causing long startup times
- Prevent multiple zero-hop tunnels in a pool
- Recognize gettext 0.20 in build scripts

**Diğer**

- Don't check clove ID in Bloom filter
- Reduce SessionTag memory usage
- Change default sig type to EdDSA for Android
- Translation updates
- Update GeoIP data
