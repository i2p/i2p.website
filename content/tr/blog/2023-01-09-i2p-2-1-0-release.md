## Güncelleme ayrıntıları

Kasım ayında yayımladığımız 2.0.0 sürümünden bu yana birkaç şey öğrendik. Router'lar bu sürüme güncellendikçe, ağda yeni SSU2 taşıma protokolümüze destek oranı yaklaşık %1'den %60'ın üzerine çıktı. Öncelikle, SSU2'nin sağlam, iyi tasarlanmış ve güvenli bir protokol olduğunu doğruladık. İkinci olarak ise, protokolün gerçekleştirmesinde küçük veya nadiren tetiklenen çok sayıda hatayı bulup düzelttik. Kümülatif olarak, bu hataların etkileri ağın performansını azalttı.

Ayrıca, ağda artan tunnel sayısının ve azalan tunnel kurulum başarı oranının farkındayız; bu durum muhtemelen Bitcoin'in yeni I2P geçici adres özelliği tarafından tetiklenmiş, ancak SSU2 hatalarımız ve diğer tıkanıklık kontrolü sorunlarıyla daha da kötüleşmiştir. I2P ağı üzerindeki talepleri azaltmak için Bitcoin ve Bitcoin dışındaki diğer projelerle birlikte çalışıyoruz. Tıkanıklık dönemlerinde ağ yükünü azaltmak için algoritmalarımızı geliştirdik. Ayrıca, ortak tıkanıklık kontrolü stratejileri geliştirmek üzere i2pd ile de işbirliği yapıyoruz.

Bu nedenle, düzeltmeleri herkese ulaştırmak için bu sürümü yaklaşık altı hafta öne çektik. i2pd geçen hafta 2.45.0 sürümünü yayımladı ve ilk sonuçlar umut verici. Yeni protokoller ve dağıtık ağların geliştirilmesi zordur. Tıkanıklık, az uyarıyla ve nedenine dair çok az ipucuyla ortaya çıkabilir. Sorunları teşhis edip umarız düzelttiğimiz sırada gösterdiğiniz sabır için teşekkür ederiz.

Her zamanki gibi, bu sürüme güncellemenizi öneririz. Güvenliği sürdürmenin ve ağa yardımcı olmanın en iyi yolu, en son sürümü çalıştırmaktır.

## Sürüm Ayrıntıları

### Changes

- Console: New status and banned peers tabs on /peers
- i2ptunnel: Add torsocks support
- i2ptunnel: Add SOCKS tunnel conversion to CONNECT outproxy
- i2ptunnel: Add SOCKS outproxy port configuration
- i2ptunnel: Update encryption type defaults
- Router: Improved congestion detection and handling
- Router: Use compressible padding for destinations and router infos (proposal 161)
- SSU: Redesign symmetric NAT detection

### Bug Fixes

- Console: Fix configuration for Argentinian Spanish
- Crypto: Fix LS2 encrypted leasesets, broken since 1.8.0
- i2psnark: Avoid OOM starting large number of torrents
- i2ptunnel: Numerous SOCKS tunnel fixes
- NTCP: Fix rare termination NPE
- Profiles: Fix profile load stopping after hitting corrupt file
- Router: Clock skew handling fixes and improvements
- SSU: Don't publish IPv4 address when configured for IPv6-only
- SSU: Fix handling of banned peers
- SSU2: Peer Test fixes and improvements
- SSU2: Termination fixes and improvements
- SSU2: Token and handshake fixes and improvements
- SSU2: Fix rare packet handling NPE
- SSU2: Fix rare termination IAE
- SSU2: Fix retransmission of session confirmed
- SSU2: Fix attempted connection to ourselves as an introducer
- UPnP: Catch rare assertion error

### Değişiklikler


- Console: Add leaseset lookup to advanced search form
- i2psnark: Add partial Dutch translation
- i2ptunnel: Allow IRCv3 ACCOUNT and CHGHOST through filter
- SSU2: Preliminary support for disabling SSU1
- Sybil: Add IPv6 address tests
- Translation updates
