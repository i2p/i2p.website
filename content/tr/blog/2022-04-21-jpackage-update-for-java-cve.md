## Güncelleme ayrıntıları

Yeni I2P Easy-Install paketleri, CVE-2022-21449 "Psychic Signatures" için bir düzeltme içeren Java Sanal Makinesi'nin en son sürümü kullanılarak oluşturulmuştur. Easy-Install paketlerini kullananların mümkün olan en kısa sürede güncelleme yapmaları önerilir. Mevcut OSX kullanıcıları güncellemeleri otomatik olarak alacaktır, Windows kullanıcıları ise kurulum programını indirme sayfamızdan indirip kurulum programını normal şekilde çalıştırmalıdır.

Linux'ta I2P router, ana sistem tarafından yapılandırılan Java Sanal Makinesi'ni (JVM) kullanır. Bu platformlardaki kullanıcıların, paket bakımcıları tarafından güncellemeler yayımlanana kadar güvenlik açığını hafifletmek için Java 14'ün altındaki kararlı bir Java sürümüne sürüm düşürmeleri önerilir. Harici bir JVM kullanan diğer kullanıcıların da JVM'yi mümkün olan en kısa sürede yamalanmış bir sürüme güncellemeleri önerilir.
