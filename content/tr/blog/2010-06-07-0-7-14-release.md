0.7.14 sürümü, her zamanki hata düzeltmeleri ve performans iyileştirmelerinden oluşan bir paket içeriyor. Ayrıca bazı i2psnark iyileştirmeleri ve eklentiler için daha fazla düzeltme var. Güncelleme yapmanız önerilir.

Lütfen ağı büyütmemize yardımcı olun. Geliştiricilere IRC #i2p üzerinden veya [forum.i2p2.de](http://forum.i2p2.de/) adresinde geri bildirim verin; [katılın](http://www.i2p2.de/getinvolved.html), herkese duyurun ve [bağış yapın](http://www.i2p2.de/donate.html)! Yeni ve mevcut çeviriler için hâlâ yardım arıyoruz. Lütfen IRC #i2p kanalında gönüllü olun.

Dosyalar [indirme sayfasında](/downloads/) mevcuttur.

Sürüm ayrıntıları:

- More bugfixes and improvements to the plugin support introduced in 0.7.12.
- I2PSnark improvements to end game handling, file listings, tunnel configuration, and translations
- Add support for throttling inbound connections in the streaming library
- Add WEBIRC support in IRC server tunnel
- Fix installer for 1.5 Java systems
- Fix bug causing multiple update downloads
- Fix bug where datagrams were not always verified when necessary
- Fix bug forcing SSU introducers when not necessary
- Fix policies for floodfill responses
- Reduce max memory used in byte caches
- Reduce job queue global locking
- More changes for performance
