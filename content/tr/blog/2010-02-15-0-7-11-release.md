0.7.11 sürümü, bir kısmı sistem saatleriyle ilgili olan ve yeni router'ların ağa hızla entegre olmasını engelleyen çeşitli sorunları giderir. Ayrıca 64-bit Windows için kurulum programını da düzeltir. Sürüm, [Jetty 5.1.15](http://old.nabble.com/jetty-5.1.15,-6.1.18-and-7.0.0.M2-td23652819.html) sürümüne, [bu güvenlik açığını](http://www.kb.cert.org/vuls/id/402580) gidermek için bir güncelleme içerir.

Her zamanki gibi, bu sürüm anonimlik ve performans açısından daha fazla iyileştirme ve çeşitli hata düzeltmeleri içeriyor. Yükseltmeniz önerilir.

Lütfen geliştiricilere IRC #i2p kanalında veya [forum.i2p2.de](http://forum.i2p2.de/) üzerinden geri bildirim verin, [katkıda bulunun](http://www.i2p2.de/getinvolved.html), duyurun ve [bağış yapın](http://www.i2p2.de/donate.html)! Hâlâ yeni ve mevcut çeviriler konusunda yardıma ihtiyacımız var. Lütfen IRC #i2p kanalında gönüllü olun.

Dosyalar [indirme sayfasında](/downloads/) mevcuttur.

Sürüm ayrıntıları:

## Yükleyici

- Fix installations on 64-bit Windows by upgrading to launch4j 3.0.1 and adding 64-bit dll to izpack.
- The wrapper is still 32-bit only. 64-bit Windows users should start I2P with the 'No Window' icon if the 'Restartable' icon does not work.

## Taşıma

- Better handle clock skews at startup, even if NTP does not work because UDP is blocked or other reasons
- Fix clock skew reporting
- Bootstrap profiles when loading the netDb from disk (broken in 0.7.8)

## Gömülü web sunucusu (Jetty)

- Upgrade to [Jetty 5.1.15](http://old.nabble.com/jetty-5.1.15,-6.1.18-and-7.0.0.M2-td23652819.html) which fixes [this vulnerability](http://www.kb.cert.org/vuls/id/402580).
- Turn on checkAliases for increased protection

## Ağ Veritabanı (netDb)

- Add some DOS protection for floodfills
- Increase the number of floodfills again

## Diğer

- Faster startup
- Bug fixes in i2psnark, BOB, clock handling, and more
