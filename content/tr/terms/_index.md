## Web Sitesinin Kullanımı

### Permitted Use

Web Sitemizi şu amaçlarla kullanabilirsiniz:

- Learn about the I2P project and software
- Download I2P software
- Access documentation and resources
- Participate in community discussions

### Prohibited Use

Web sitemizi şu amaçlarla kullanamazsınız:

- Violate any applicable laws or regulations
- Transmit malicious code or harmful software
- Attempt to gain unauthorized access to our systems
- Impersonate others or misrepresent your affiliation
- Scrape or harvest data in violation of our robots.txt
