Vydání 0.7.11 opravuje několik problémů, některé související se systémovými hodinami, které bránily novým routers v rychlém začlenění do sítě. Zároveň opravuje instalační program pro 64bitové Windows. Vydání také zahrnuje aktualizaci na [Jetty 5.1.15](http://old.nabble.com/jetty-5.1.15,-6.1.18-and-7.0.0.M2-td23652819.html) za účelem opravy [této zranitelnosti](http://www.kb.cert.org/vuls/id/402580).

Jako obvykle toto vydání přináší další vylepšení anonymity a výkonu a také řadu oprav chyb. Doporučujeme aktualizovat.

Prosíme, dejte vývojářům zpětnou vazbu na IRC #i2p nebo na [forum.i2p2.de](http://forum.i2p2.de/) a [zapojte se](http://www.i2p2.de/getinvolved.html), šiřte povědomí a [přispějte](http://www.i2p2.de/donate.html)! Stále hledáme pomoc s novými i stávajícími překlady. Prosíme, přihlaste se jako dobrovolník na IRC #i2p.

Soubory jsou k dispozici na [stránce ke stažení](/downloads/).

Podrobnosti vydání:

## Instalátor

- Fix installations on 64-bit Windows by upgrading to launch4j 3.0.1 and adding 64-bit dll to izpack.
- The wrapper is still 32-bit only. 64-bit Windows users should start I2P with the 'No Window' icon if the 'Restartable' icon does not work.

## Přenos

- Better handle clock skews at startup, even if NTP does not work because UDP is blocked or other reasons
- Fix clock skew reporting
- Bootstrap profiles when loading the netDb from disk (broken in 0.7.8)

## Vestavěný webový server (Jetty)

- Upgrade to [Jetty 5.1.15](http://old.nabble.com/jetty-5.1.15,-6.1.18-and-7.0.0.M2-td23652819.html) which fixes [this vulnerability](http://www.kb.cert.org/vuls/id/402580).
- Turn on checkAliases for increased protection

## Síťová databáze

- Add some DOS protection for floodfills
- Increase the number of floodfills again

## Ostatní

- Faster startup
- Bug fixes in i2psnark, BOB, clock handling, and more
