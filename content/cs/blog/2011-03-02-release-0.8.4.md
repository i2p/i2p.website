Vydání 0.8.4 obsahuje některá vylepšení výkonu a důležité opravy chyb. Také i2psnark nyní podporuje magnet odkazy. Jako obvykle se doporučuje aktualizace.

Pomozte, prosím, rozšířit síť. [Zapojte se](http://www.i2p2.de/getinvolved.html), šiřte povědomí a [přispějte](http://www.i2p2.de/donate.html)! Pokud najdete chybu, prosím, nahlaste ji na [gitlab](http://i2pgit.org/i2p-hackers/i2p.i2p/issues). Stále hledáme pomoc s novými i stávajícími překlady. Přihlaste se prosím jako dobrovolníci na IRC #i2p-dev.

Soubory jsou k dispozici na [stránce ke stažení](/downloads/).

## Podrobnosti o vydání:

- Fix a severe memory leak in router I2CP session management that caused router crashes for people running the Robert bittorrent client
- Fix a bug from 0.8.2 that filtered cookies in the HTTP Server tunnel, causing authentication problems for some eepsites (I2P Sites)
- Several fixes for rare NPEs

## I2PSnark

- Magnet and "maggot" link support
- Peer Exchange
- Metadata transfer
- Details page enhancements
- Bandwidth limiter improvements
- Bug fixes

## Ostatní

- More performance improvements
- Adjustment to peer selection, to hopefully use more of the available network capacity
- Translation updates
- Start of a new Arabic translation by 'hamada'
