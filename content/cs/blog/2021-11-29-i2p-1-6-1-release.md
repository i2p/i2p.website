## Podrobnosti aktualizace

Poznámka: 1.6.1 je aktuální verze. 1.6.0 byla označena tagem, ale nebyla vydána.

Toto vydání dokončuje nasazení dvou zásadních aktualizací protokolu vyvinutých v roce 2021. Přechod na šifrování X25519 pro routers je urychlen a očekáváme, že téměř všechny routers budou do konce roku překlíčovány. Krátké tunnel build messages jsou povoleny pro významné snížení využití šířky pásma.

Do nového instalačního průvodce jsme přidali panel pro výběr motivu. Zlepšili jsme výkon SSU a opravili problém se zprávami SSU typu peer test. Bloomův filtr používaný při vytváření tunnel byl upraven, aby se snížila spotřeba paměti. Zlepšili jsme podporu pro zásuvné moduly v jiných jazycích než Java.

Kromě toho dosahujeme vynikajícího pokroku v návrhu našeho nového UDP transportu SSU2 a očekáváme, že s implementací začneme začátkem příštího roku.

Jako obvykle doporučujeme, abyste aktualizovali na tuto verzi. Nejlepším způsobem, jak zachovat bezpečnost a pomoci síti, je používat nejnovější verzi.

## PODROBNOSTI VYDÁNÍ

**Změny**

- Console: Add theme selection to new-install wizard
- Jetty 9.3.30.v20211001
- Router: Increase probability of rekey to X25519
- SSU: Performance improvements
- SSU: Improve security of peer test
- Tomcat 9.0.54
- Tunnels: Enable sending new short build messages

**Opravy chyb**

- i2ptunnel: Fix encryption type selection logic

**Ostatní**

- Improve support for non-Java plugins
- Router: Reduce tunnel build Bloom filter memory usage
- Updates: More changes to support bundled updaters
- Translation updates
