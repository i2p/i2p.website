## Podrobnosti aktualizace

Toto vydání obsahuje opravy chyb v i2psnarku, routeru, I2CP a UPnP. Opravy routeru se týkají chyb spojených se soft restartem, IPv6, testováním peerů v SSU, ukládáním do síťové databáze a sestavováním tunnelů. Zpracování rodin routerů a klasifikace Sybil byly také výrazně vylepšeny.

Spolu s i2pd vyvíjíme náš nový UDP transport, SSU2. SSU2 přinese významná zlepšení výkonu i bezpečnosti. Umožní nám také konečně nahradit naše poslední použití velmi pomalého šifrování ElGamal, čímž dokončíme kompletní modernizaci kryptografie, kterou jsme zahájili přibližně před 9 lety. Toto vydání obsahuje předběžnou implementaci, která je ve výchozím nastavení vypnutá. Pokud se chcete zapojit do testování, aktuální informace hledejte na zzz.i2p.

Jako obvykle doporučujeme aktualizovat na tuto verzi. Nejlepším způsobem, jak udržovat bezpečnost a pomoci síti, je používat nejnovější verzi.

## Podrobnosti o vydání

**Změny**

- SSU2: Preliminary implementation
- Tomcat 9.0.62

**Opravy chyb**

- Crypto: Fix handling of EdDSA certs with official OID
- I2CP: Fix external connections when session ID is 0
- I2PSnark: Fix size calculation causing tracker errors
- I2PSnark standalone: More fixes and improvements
- Router: Family fixes and improvements
- Router: Fix database store logic
- Router: Fix invalid store and rekeying at startup
- Router: Don't use he.net addresses for GeoIP
- Router: Soft restart fixes
- SSU: Peer test fixes
- SSU: Publish empty IPv6 address when missing introducers
- SSU: Reduce ack delay to improve performance
- Transport: Fix UPnP deadlock
- Tunnels: Don't build client tunnels shorter than minimum length
- UPnP: IPv6 fixes

**Ostatní**

- API version: 0.9.54
- Console: Setup wizard redesign
- i2psnark: Load system mime types if available
- I2PSnark standalone: Add system tray
- Router: Reduce build reply timeout so we will retry faster
- Router: Avoid Sybil penalty for trusted large families
- Source: Remove BOB
- Translation updates
