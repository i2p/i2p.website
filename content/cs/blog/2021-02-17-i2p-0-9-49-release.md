## Podrobnosti aktualizace

Verze 0.9.49 pokračuje v práci na tom, aby bylo I2P rychlejší a bezpečnější. Máme několik vylepšení a oprav pro transport SSU (UDP), které by měly vést k vyšším rychlostem. Toto vydání také zahajuje migraci na nové, rychlejší šifrování ECIES-X25519 pro routery. (Destinace toto šifrování používají už několik vydání) Na specifikacích a protokolech pro nové šifrování pracujeme už několik let a blížíme se ke konci! Migrace potrvá několik vydání, než bude dokončena.

V tomto vydání, aby se minimalizovalo narušení, budou nové instalace a jen velmi malé procento stávajících instalací (náhodně vybraných při restartu) používat nové šifrování. Pokud váš router provede „rekey“ (vygeneruje nové klíče) pro používání nového šifrování, může mít několik dní po restartu nižší objem provozu nebo nižší spolehlivost než obvykle. To je normální, protože váš router si vygeneroval novou identitu. Výkon by se měl po nějaké době vrátit na obvyklou úroveň.

Už dříve jsme dvakrát "rekeyed" (znovu vygenerovali klíče) síť, když jsme měnili výchozí typ podpisu, ale toto je poprvé, co jsme změnili výchozí typ šifrování. Doufejme, že vše proběhne hladce, ale začínáme pomalu, abychom měli jistotu.

Jako obvykle doporučujeme, abyste aktualizovali na tuto verzi. Nejlepším způsobem, jak udržovat bezpečnost a pomoci síti, je provozovat nejnovější verzi.

**Změny**

- Build: Git migration

- i2psnark WebSeed support
- i2psnark padding file support
- i2ptunnel: Move proxy resources to jar
- Router: Redesign ECIES encryption for floodfills (proposal 156)

- Router: Reduce Sybil threshold
- Router: ECIES for new routers
- Router: Start of ECIES migration
- SSU: Send individual fragments of messages
- SSU: Westwood+ congestion control
- SSU: Fast retransmit

- Build: Fix Gradle build

- I2CP: Fix encrypted leaseset combined with ECIES crypto or offline keys
- i2ptunnel: Fix config file saving issues
- Router: Fix leaseset request fails causing watchdog to bark
- Router: Hidden mode fixes
- SSU: Fix partial acks not being sent
- SSU: Fix occasional high CPU usage

- Crypto: AES performance improvements

- i2ptunnel: Remove DSA shared clients
- Proxy: Add jump servers

- Router: Tunnel peer selection changes

- Router: Verify RI stores for a while after startup
