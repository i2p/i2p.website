Vydání I2P 0.6.1.32
=====================

Vydání 0.6.1.32 obsahuje několik oprav chyb, nový algoritmus budování tunnel od zzz, který reguluje intenzitu budování tunnel podle předchozích úrovní úspěšnosti budování tunnel, a další zlepšení výkonu.

Toto vydání je zpětně kompatibilní a pokud už máte nainstalovanou verzi 0.6.1.31, ověří podpis na podepsaném aktualizačním souboru (takže můžete aktualizovat automaticky).

Abychom však zajistili lepší rozložení zátěže mezi našimi 3 podepsanými aktualizačními servery, doporučuji provést následující volitelnou kontrolu:

* Open the update configuration page on:

http://localhost:7657/configupdate.jsp

* If you see 3 update URLs, no action is needed.

Pokud vidíte jednu aktualizační adresu URL, otevřete stránku pokročilého nastavení na http://localhost:7657/configadvanced.jsp a smažte položku konfigurace začínající na "router.updateURL=". Poté klikněte na "Apply" a u aktualizačních adres URL by se měla projevit výchozí konfigurace.

* Please note that since 0.6.1.31, configuration entries

může zabírat více řádků, proto si prosím dvakrát ověřte, kde začíná další položka, abyste ji nepoškodili!
