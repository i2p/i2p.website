## Podrobnosti aktualizace

Změny v tomto vydání jsou mnohem výraznější než obvykle! Přepracovali jsme konzoli routeru, aby byla snáze pochopitelná, zlepšili jsme přístupnost a podporu napříč prohlížeči a celkově jsme ji zpřehlednili. Je to první krok v dlouhodobém plánu, jak učinit konzoli routeru uživatelsky přívětivější. Také jsme do i2psnarku přidali podporu hodnocení a komentářů torrentů.

As usual, we recommend that you update to this release. The best way to maintain security and help the network is to run the latest release.
