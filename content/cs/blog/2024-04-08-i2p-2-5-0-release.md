Toto vydání, I2P 2.5.0, přináší více vylepšení viditelných pro uživatele než vydání 2.4.0, které bylo zaměřeno na implementaci strategie izolace NetDB.

Do I2PSnarku byly přidány nové funkce, například možnost vyhledávat v torrentech. Byly opraveny chyby, které zlepšují kompatibilitu s dalšími I2P torrentovými klienty, jako jsou BiglyBT a qBittorrent. Rádi bychom poděkovali všem vývojářům, kteří pracovali s libtorrent a qBittorrent, aby umožnili podporu I2P a dále ji vylepšili. Nové funkce byly přidány také do SusiMailu, včetně podpory formátování Markdown v e-mailech a možnosti vkládat do e-mailů přílohy přetažením. Tunnels vytvořené pomocí správce skrytých služeb nyní podporují "Keepalive", což zlepšuje výkon a kompatibilitu s webovými technologiemi a umožňuje sofistikovanější I2P stránky.

V rámci tohoto vydání jsme také provedli několik úprav v NetDB, abychom zvýšili odolnost vůči spamu a zlepšili schopnost routeru odmítat podezřelé zprávy. To bylo součástí snahy "auditovat" implementaci obranných opatření "Sub-DB isolation" (izolace dílčí databáze) z vydání 2.4.0. Toto šetření odhalilo jedno drobné narušení izolace, které jsme opravili. Tento problém byl interně objeven a opraven týmem I2P.

Během tohoto vydání bylo provedeno několik vylepšení procesu vydávání našich downstream distribucí (odvozené distribuce) pro Android a Windows. To by mělo vést ke zlepšení doručování a dostupnosti těchto downstream produktů.

As usual, we recommend that you update to this release. The best way to maintain security and help the network is to run the latest release.

## Podrobnosti o vydání

### Changes

- I2PTunnel: Implement support for Keepalive/Server-side Persistence
- Susimail: Add markdown support for formatted plain-text content
- Susimail: Add HTML Email support
- I2PSnark: Add search capability
- I2PSnark: Preserve private=0 in torrent files
- Data: Store compressed RI and LS

### Změny

- Susimail: Fix handling of forwarded mail with attachments
- Susimail: Fix handling of forwarded mail with unspecified encoding
- Susimail: Fix forwarding of HTML-only email
- Susimail: Bugfixes in presentation of encoded attachmments, mail body
- I2PSnark: Handle data directory changes
- SSU2: Cancel peer test if Charlie does not have B cap
- SSU2: Treat peer test result as unknown if Charlie is unreachable
- Router: Filter additional garlic-wrapped messages
- I2CP: Prevent loopback messages to same session
- NetDB: Resolve Exploratory/Router isolation-piercing event

### Opravy chyb

- API 0.9.62
- Translation updates
