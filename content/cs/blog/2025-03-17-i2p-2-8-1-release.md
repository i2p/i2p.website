Verze 2.8.1 opravuje přístupy k lokálním stránkám, které v 2.8.0 nefungovaly. Do několika aplikací v konzoli jsme přidali notifikační bubliny, aby zvýraznily zprávy aplikací. Odstranili jsme chybu v testu tunnelu, která mohla ovlivňovat spolehlivost tunnelů. Adresář nyní má novou kartu "latest", která zobrazuje nejnovější názvy hostitelů přidané z vašich odběrů. Došlo k několika opravám a vylepšením tmavého motivu.

We fixed the installer that previously failed on Java 21 or higher. We also fixed installing to directory paths that contain spaces on Linux. For those of you that use Docker, we updated outdated Docker container and container libraries.

Jako obvykle doporučujeme, abyste aktualizovali na tuto verzi. Nejlepší způsob, jak zachovat bezpečnost a pomoci síti, je používat nejnovější verzi.

## Podrobnosti o vydání

### Changes

- New console notification bubbles
- New addressbook sort-by-latest tab
- Add support for .i2p.alt hostnames (RFC 9476)

### Změny

- Fix local site access bug
- Fix installer failure on Java 21+ (IzPack 5.2.4)
- Fix tunnel tests for outbound client tunnels
- Fix installing to paths with spaces (non-Windows)
- Console dark theme fixes and improvements
- Fix AIOOBE on large signed streaming packets
- Fix compression of Router Identities
- Update outdated Docker container and container libraries

### Opravy chyb

- Add logo to first installer panel
- Add progress bar to installer panels
- Use SHA256 pool in Noise
- Move certs page to a debug page tab
- Reduce memory usage in AddressBean
- Set bulk profile for servers and i2psnark
- Prep for tunnel bandwidth parameters (proposal 168)
- Proxy error page improvements
- Check key order when parsing RI mappings
- Reduce i2psnark peer check interval
- Translation updates

[Úplný seznam opravených chyb](http://git.idk.i2p/i2p-hackers/i2p.i2p/-/issues?scope=all&state=closed&milestone_title=2.9.0)
