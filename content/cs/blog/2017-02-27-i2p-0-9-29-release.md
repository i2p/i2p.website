## Podrobnosti aktualizace

Verze 0.9.29 obsahuje opravy pro řadu tiketů v Tracu, včetně dočasných řešení pro poškozené komprimované zprávy. Nyní podporujeme NTP přes IPv6. Přidali jsme předběžnou podporu Dockeru. Nyní máme přeložené man stránky. Nyní přes HTTP proxy propouštíme hlavičky Referer stejného původu. Přibyly další opravy pro Javu 9, ačkoli zatím nedoporučujeme Javu 9 pro běžné použití.

As usual, we recommend that you update to this release. The best way to maintain security and help the network is to run the latest release.
