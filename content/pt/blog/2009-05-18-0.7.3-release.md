I2P versão 0.7.3
==================

A versão 0.7.3 do I2P contém várias correções de bugs e melhorias, inclui muito trabalho de manutenção e novas versões dos protocolos de gateway de aplicação SAM e BOB.

Um patch para o servidor web interno do I2P, o Jetty, está incluído para corrigir o comportamento do SusiDNS no Windows. A forma como os nós do I2P se oferecem para participar do Network Database foi alterada para evitar que muitos façam isso simultaneamente, e a criação de novos tunnels participantes agora é limitada antes que mensagens nos tunnels existentes comecem a ser descartadas.

Várias melhorias na Router Console e em outros componentes estão incluídas; o I2PSnark pode lidar com torrents maiores e mais arquivos, enquanto o trabalho continua na interface de desktop experimental. Recomenda-se a atualização.
