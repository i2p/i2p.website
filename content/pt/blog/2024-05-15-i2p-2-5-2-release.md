I2P 2.5.2 foi lançado para corrigir um bug introduzido na 2.5.0 que causava o truncamento de parte do conteúdo HTTP.

Como de costume, recomendamos que você atualize para esta versão. A melhor forma de manter a segurança e ajudar a rede é executar a versão mais recente.

## Detalhes da versão

### Changes

- Console: Update rrd4j to 3.9.1-preview
- Router: Publish G cap if symmetric natted

### Alterações

- i2ptunnel: Fix bug causing truncation of some HTTP content
- i2ptunnel: Fix custom option form width (light theme)
- Tunnels: Fix selection of peers with expired RIs

### Correções de bugs

- Translation updates

[Lista completa de bugs corrigidos](http://git.idk.i2p/i2p-hackers/i2p.i2p/-/issues?scope=all&state=closed&milestone_title=2.5.2)
