Olá I2P! Este é o primeiro lançamento assinado por mim (str4d), após 49 lançamentos assinados por zzz. Este é um teste importante da nossa redundância em tudo, incluindo pessoas.

## Manutenção

Minha chave de assinatura tem sido incluída nas atualizações do router há mais de dois anos (desde a 0.9.9), portanto, se você estiver em uma versão recente do I2P, esta atualização deve ser tão fácil quanto qualquer outra. No entanto, se você estiver executando uma versão anterior à 0.9.9, primeiro será necessário atualizar manualmente para uma versão recente. Depois de atualizar manualmente, seu router encontrará e baixará a atualização 0.9.23 como de costume.

Se você instalou o I2P por meio de um gerenciador de pacotes, não é afetado pela mudança e pode atualizar normalmente.

## Detalhes da atualização

A migração de RouterInfos para novas e mais fortes assinaturas Ed25519 está indo bem, com a estimativa de que pelo menos metade da rede já tenha trocado suas chaves. Esta versão acelera o processo de troca de chaves. Para reduzir a rotatividade na rede, seu router terá uma pequena probabilidade de converter para Ed25519 a cada reinicialização. Quando ele trocar de chaves, espere ver um menor uso de largura de banda por alguns dias enquanto se reintegra à rede com sua nova identidade.

Observe que esta será a última versão a oferecer suporte ao Java 6. Atualize para o Java 7 ou 8 o mais rápido possível. Já estamos trabalhando para tornar o I2P compatível com o próximo Java 9, e parte desse trabalho está nesta versão.

Também fizemos algumas melhorias menores no I2PSnark e adicionamos uma nova página no routerconsole para visualizar itens de notícias mais antigos.

Como de costume, recomendamos que atualize para esta versão. A melhor maneira de manter a segurança e ajudar a rede é executar a versão mais recente.
