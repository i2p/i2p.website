Como todos vocês notaram, o crescimento sem precedentes da rede a partir de 5 de outubro aumentou dramaticamente o congestionamento da rede, especialmente à noite (UTC) e nos fins de semana. As duas últimas versões incluíram algumas alterações que esperávamos que aliviassem a pressão, mas, infelizmente, essas medidas tiveram apenas resultados modestos. O principal problema é limitar o número de conexões diretas de router para router na rede. Isso não é um problema novo; temos trabalhado nisso há vários anos, com bons resultados. No entanto, o crescimento recente nos levou além do limite mais uma vez.

A versão 0.8.11 inclui várias alterações adicionais para reduzir o número de conexões de router para router e aumentar a capacidade de conexões e de construção de tunnel. O objetivo, claro, é melhorar as taxas de sucesso na construção de tunnel e a confiabilidade geral. Como sempre, há algumas correções de bugs e atualizações de traduções.

Damos as boas-vindas a todos os nossos novos usuários. Pedimos paciência enquanto trabalhamos para melhorar o desempenho da rede. Depurar problemas de congestionamento em uma rede anônima distribuída é um desafio contínuo. Ajude a melhorar a rede reiniciando seu router assim que a atualização for baixada.

Os arquivos estão disponíveis na [página de download](/downloads).

## DETALHES DA VERSÃO

## Principais mudanças

- Improve peer profile capacity metric to include factors for connection, reachability, country, and router hash, to reduce connection churn
- Adjust connection limits and timeouts to increase capacity
- Adjust router info and peer profile expirations to reduce memory usage
- Rebuild existing tunnels some of the time to reduce connection churn and improve build success
- Build Executor / Handler thread separation to increase build handling capacity for high-speed routers
- Bloom Filter optimizations to reduce lock contention for high-speed routers
- SSU introducer changes

## Correções de erros

- Fix expiration of peer profiles

## Outros

- Ukrainian translation updates
