Lançamento do I2P 0.6.5
====================

A versão 0.6.5 introduz novos componentes, remove alguns antigos (como o antigo transporte TCP) e foi otimizada para ter melhor desempenho.

O protocolo BOB (Basic Open Bridge) é introduzido, para uso por aplicações cliente que não podem importar diretamente as bibliotecas I2CP. Isso torna obsoleto o antigo protocolo SAM, que era anteriormente usado nesses casos. Por enquanto, porém, o BOB ainda não é iniciado automaticamente em novas instalações, e o SAM permanece ativo em instalações antigas.

O código aprimorado deve ser melhor em prevenir congestionamento ao descartar probabilisticamente tráfego participante e também se comportar melhor quando ocorrer congestionamento. O floodfill NetDB deve operar de forma mais confiável, a biblioteca de streaming deve escolher tamanhos de mensagem melhores, oferecer uma função de timeout de socket, e o trabalho prossegue no modo de operação "hidden" para routers I2P.

- From this release onward, I2P requires Java 1.5 or higher.

Se você não tem certeza sobre a sua versão do Java, você pode verificar abrindo uma janela de terminal ou prompt de comando e digitando o comando "java -version". Se você tiver uma versão mais antiga do Java instalada, atualize-a primeiro!

Esta versão é retrocompatível: se você já tem a 0.6.1.31 ou superior instalada, ela reconhecerá a assinatura no arquivo de atualização assinado e você poderá atualizar automaticamente.
