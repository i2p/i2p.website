0.9.14.1 inclui correções para o formulário "Adicionar Torrent" no i2psnark e alguns outros formulários da web. Restauramos a capacidade de instalar plugins pelo console, mas você deve primeiro editar o arquivo router.config (em ~/.i2p/ ou /var/lib/i2p/i2p-config/ ou %APPDATA%\I2P\) para adicionar a linha routerconsole.enablePluginInstall=true. Outros recursos avançados raramente usados que foram removidos na 0.9.14 podem ser restaurados adicionando a linha routerconsole.advanced=true.

Como de costume, se configurado com o padrão "Download and Verify", o router fará o download da atualização automaticamente e exibirá um botão para reiniciar. No entanto, devido a um bug na versão 0.9.14, se a sua atualização estiver configurada para "Notify only", o botão de download não será exibido. Você deve alterar sua configuração para "Download and Verify" ou "Download, Verify, and Restart" para atualizar.

Se você ainda estiver executando a versão 0.9.13 ou anterior, recomendamos que atualize para esta versão o quanto antes. Se você não verifica o console do router com frequência, por favor, considere alterar sua configuração para "Download, Verify, and Restart" para garantir que você esteja sempre executando a versão mais recente.

## DETALHES DA VERSÃO

### Bug Fixes

- Fix i2psnark add torrent form
- Fix iptunnel custom options form
- Fix update download buttons

### Correções de erros

- Restore all console features if routerconsole.advanced=true
- Restore plugin install if routerconsole.enablePluginInstall=true
- Restore client adds/changes if routerconsole.enableClientChange=true
- Plugin signing keys are now whitelisted unless routerconsole.allowUntrustedPlugins=true
- More escaping and cleanups in forms and messages
- Update GeoIP data (new installs and PPA only)
