## Detalhes da atualização

0.9.39 inclui alterações abrangentes para novos tipos de banco de dados de rede (proposta 123). O plugin i2pcontrol agora é distribuído como uma aplicação web para dar suporte ao desenvolvimento de aplicações RPC. Vários erros importantes em streaming, i2psnark e NTCP2 foram corrigidos.

Como de costume, recomendamos atualizar para esta versão. A melhor maneira de manter a segurança e ajudar a rede é executar a versão mais recente.
