## Detalhes da atualização

0.9.29 contém correções para inúmeros tickets do Trac, incluindo soluções alternativas para mensagens comprimidas corrompidas. Agora oferecemos suporte a NTP sobre IPv6. Adicionamos suporte preliminar ao Docker. Agora temos páginas de manual (man pages) traduzidas. Agora encaminhamos cabeçalhos Referer de mesma origem através do proxy HTTP. Há mais correções para o Java 9, embora ainda não recomendemos o Java 9 para uso geral.

Como de costume, recomendamos que atualize para esta versão. A melhor forma de manter a segurança e ajudar a rede é executar a versão mais recente.
