## Detalhes da atualização

0.9.45 contém correções importantes para o modo oculto e o testador de largura de banda. Há uma atualização para o tema escuro do console. Continuamos trabalhando na melhoria do desempenho e no desenvolvimento de uma nova criptografia de ponta a ponta (proposta 144).

Como de costume, recomendamos que atualize para esta versão. A melhor maneira de manter a segurança e ajudar a rede é executar a versão mais recente.

## DETALHES DA VERSÃO

**Alterações**

- Dark theme improvements
- Console icon changes
- Move some translations from console to router and core for embedded uses
- Jetty 9.2.29
- Tomcat 8.5.50
- i2psnark HTML5 preview player for partial files

**Correções de bugs**

- Numerous bandwidth test fixes
- Hidden mode fixes to prevent losing peers
- Hebrew translation fixes

**Outros**

- Content Security Policy improvements
- SSU extend timeout for large messages
- ECIES-X25519 continued development and fixes
- Cancel timeout jobs on message reply
- Add SSL and SSL redirect support to EepGet
- Translation updates
- Update JBigI library

[Lista completa de bugs corrigidos](http://trac.i2p2.i2p/query?resolution=fixed&milestone=0.9.45)
