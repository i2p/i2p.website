Este glossário define termos comuns utilizados na documentação e discussões do I2P.

## B

**Blockchain** Um livro-razão distribuído que pode registrar transações entre múltiplas partes de forma eficiente, verificável e permanente.

## D

**Descentralização** Arquitetura de rede que evita dependência de uma única parte. Abrange tecnologias peer-to-peer, blockchain, federadas e distribuídas.

**Destination** A identidade criptográfica de um túnel. Estas são as identidades de clientes e servidores dentro da rede I2P.

**DHT (Distributed Hash Table)** Utilizado em alguns projetos para conectar peers entre si através do armazenamento de informações na forma de pares chave-valor de maneira distribuída.

**Sistemas Distribuídos** Tópico acadêmico dentro da Ciência da Computação relacionado ao design de sistemas de computadores que consistem em muitos computadores individuais conectados através de uma rede.

## F

**Federado** Federação permite que implementações separadas de um serviço se comuniquem entre si através de um protocolo comum.

## H

**Hash** Um número, geralmente exibido como uma sequência de letras e números. Pode servir como uma "impressão digital" que identifica dados de forma única.

## I

**I2P** Invisible Internet Project: um projeto destinado a fornecer uma camada de anonimato, para que os usuários possam se comunicar anonimamente usando uma variedade de aplicações.

**Endereço IP** Um número de um computador ou rede que é único e, portanto, pode ser usado para endereçá-lo.

## L

**LeaseSet** Uma coleção de informações necessárias para comunicar com um cliente ou servidor num Destination específico.

## P

**Peer-to-Peer (P2P)** Os peers disponibilizam uma parte dos seus recursos diretamente a outros participantes da rede, sem a necessidade de coordenação central por servidores ou hosts estáveis.

## R

**Router** O software central do I2P, que roteia pacotes criptografados na rede I2P.

**RouterIdentity** Uma coleção de informações necessárias para comunicar diretamente com um router.

## T

**TCP/UDP** Os dois protocolos de transporte fundamentais usados na Internet.

**Tunnel** Um caminho de comunicação anônimo entre um cliente ou servidor e a rede I2P.

## U

**UX** Experiência do usuário (User experience), a experiência geral de uma pessoa ao usar um produto ou serviço.

## W

**WebRTC** Um padrão de protocolo para estabelecer conexões em um navegador web onde os dados passam diretamente entre os usuários.
