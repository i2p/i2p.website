## Requisitos do Sistema

Para executar o I2P, você precisará de:

- **Java 8** ou superior. Recomendamos a versão mais recente do OpenJDK.
- Pelo menos **512 MB de RAM** disponível para o I2P.
- **Conexão de internet** estável.

## Instruções de Instalação

1. **Baixe o instalador** do I2P a partir do [site oficial](https://geti2p.net).
2. **Execute o instalador** e siga as instruções na tela.
3. Após a instalação, **inicie o serviço** I2P através do comando `i2prouter start`.

## Configuração Inicial

Após iniciar o I2P, acesse o **console do router** em `http://127.0.0.1:7657`. Aqui você pode:

- Verificar o status da rede.
- Configurar **túneis** e **eepsites**.
- Ajustar as configurações de **garlic encryption**.

## Conectividade

O I2P utiliza os protocolos **NTCP2** e **SSU** para comunicação. Certifique-se de que sua rede permite o tráfego nesses protocolos.

android: ### Configuração do I2P

Para configurar o I2P, siga os passos abaixo:

1. **Instalação**: Baixe o pacote de instalação do I2P do [site oficial](https://geti2p.net). Siga as instruções para instalar o software no seu sistema operacional.

2. **Inicialização do Router**: Após a instalação, inicie o router I2P. Isso pode ser feito através do comando `i2prouter start` no terminal.

3. **Configuração do Túnel**: Acesse a interface de configuração do I2P através do endereço `http://127.0.0.1:7657`. Navegue até a seção de configuração de túnel para criar e gerenciar túneis.

4. **Verificação do NetDb**: Certifique-se de que o netDb (base de dados de rede) está atualizado. Isso garante que o router tenha as informações mais recentes sobre outros peers na rede.

5. **Configuração de Serviços**: Configure serviços adicionais, como o I2PTunnel, para acessar eepsites ou outros serviços ocultos.

6. **Monitoramento**: Use a interface web para monitorar o desempenho do router e ajustar as configurações conforme necessário.

### Segurança e Privacidade

- **Criptografia Garlic**: O I2P utiliza criptografia garlic para proteger os dados transmitidos. Isso garante que as comunicações sejam privadas e seguras.
- **Atualizações Regulares**: Mantenha o software I2P atualizado para proteger contra vulnerabilidades conhecidas.
- **Configuração de Firewall**: Configure o firewall para permitir o tráfego do I2P, garantindo que o router possa se conectar a outros peers.
