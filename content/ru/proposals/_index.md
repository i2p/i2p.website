# Предложения по улучшению I2P

Этот раздел содержит все предложения по улучшению I2P (IEP) - официальные документы, которые описывают изменения, улучшения и усовершенствования сети I2P и ее спецификаций.

## О процессе предложений

Предложения I2P следуют структурированному процессу, адаптированному из процесса предложений Tor. Каждое предложение проходит несколько этапов:

- **Открыто**: В стадии обсуждения
- **Принято**: Готово к реализации
- **Завершено**: Принято и реализовано
- **Закрыто**: Объединено с основными спецификациями
- **Отклонено**: Не будет реализовано
