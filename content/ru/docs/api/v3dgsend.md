## Обзор

`v3dgsend` — это утилита командной строки для отправки I2P-датаграмм через интерфейс SAMv3. Она полезна для тестирования доставки датаграмм, создания прототипов сервисов и проверки сквозного поведения без написания полноценного клиента.

Типичные варианты использования включают:

- Дымовое тестирование достижимости датаграмм до Destination
- Проверка конфигурации файрвола и адресной книги
- Эксперименты с сырыми и подписанными (с возможностью ответа) датаграммами

## Использование

Базовый способ запуска зависит от платформы и способа установки. Общие варианты включают:

- Destination: base64 Destination или имя `.i2p`
- Protocol: raw (PROTOCOL 18) или signed (PROTOCOL 17)
- Payload: встроенная строка или входной файл

Обратитесь к документации пакетов вашего дистрибутива или к выводу `--help` для получения точных флагов.
