Summer Dev в самом разгаре: мы налаживаем процессы, сглаживаем острые углы и наводим порядок. Настало время нашего первого дайджеста, в котором мы введём вас в курс дела и расскажем, как продвигается работа!

## Месяц API

Наша цель на этот месяц заключалась в том, чтобы "вписаться" — сделать так, чтобы наши API и библиотеки работали в рамках существующей инфраструктуры различных сообществ, чтобы разработчики приложений могли работать с I2P более эффективно, а пользователям не приходилось беспокоиться о деталях.

### Java / Android

Клиентские библиотеки I2P теперь доступны в Maven Central! Это должно заметно упростить использование I2P в приложениях Java-разработчиков. Вместо того чтобы получать библиотеки из текущей установки, они могут просто добавить I2P в свои зависимости. Обновление до новых версий также станет значительно проще.

Клиентская библиотека I2P Android также была обновлена для использования новых библиотек I2P. Это означает, что кроссплатформенные приложения могут работать нативно как с I2P Android, так и с настольным I2P.

### Java / Android

#### txi2p

Плагин Twisted `txi2p` теперь поддерживает порты внутри I2P и будет прозрачно работать поверх локальных, удалённых и доступных через переадресацию портов SAM API. См. документацию для получения инструкций по использованию и сообщайте о любых проблемах на GitHub.

#### i2psocket

Вышла первая (бета) версия `i2psocket`! Это прямая замена стандартной библиотеке Python `socket`, добавляющая поддержку I2P через SAM API. См. страницу на GitHub для инструкций по использованию и для сообщения о проблемах.

Мы рады сотрудничать с Tahoe-LAFS в июле! В I2P уже долгое время функционирует один из крупнейших общедоступных гридов, использующий модифицированную версию Tahoe-LAFS. Во время Месяца приложений мы будем помогать им в текущей работе по добавлению встроенной поддержки I2P и Tor, чтобы пользователи I2P могли воспользоваться всеми улучшениями upstream (основного проекта).

Есть несколько других проектов, с которыми мы будем обсуждать их планы по интеграции с I2P и помогать с проектированием. Следите за обновлениями!
