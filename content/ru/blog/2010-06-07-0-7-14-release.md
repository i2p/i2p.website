Релиз 0.7.14 содержит обычный набор исправлений ошибок и оптимизаций производительности. Также имеются некоторые улучшения i2psnark и дополнительные исправления для плагинов. Рекомендуется обновление.

Пожалуйста, помогите развивать сеть. Дайте разработчикам обратную связь в IRC #i2p или на [forum.i2p2.de](http://forum.i2p2.de/), [присоединяйтесь](http://www.i2p2.de/getinvolved.html), распространяйте информацию и [пожертвуйте](http://www.i2p2.de/donate.html)! Мы всё ещё ищем помощь в работе над новыми и существующими переводами. Пожалуйста, присоединяйтесь как волонтёр в IRC #i2p.

Файлы доступны на [странице загрузки](/downloads/).

Сведения о выпуске:

- More bugfixes and improvements to the plugin support introduced in 0.7.12.
- I2PSnark improvements to end game handling, file listings, tunnel configuration, and translations
- Add support for throttling inbound connections in the streaming library
- Add WEBIRC support in IRC server tunnel
- Fix installer for 1.5 Java systems
- Fix bug causing multiple update downloads
- Fix bug where datagrams were not always verified when necessary
- Fix bug forcing SSU introducers when not necessary
- Fix policies for floodfill responses
- Reduce max memory used in byte caches
- Reduce job queue global locking
- More changes for performance
