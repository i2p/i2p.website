Релиз I2P 0.6.1.32
====================

Релиз 0.6.1.32 содержит несколько исправлений ошибок, новый алгоритм построения tunnel (туннель) от zzz, который регулирует интенсивность построения tunnel в зависимости от предыдущих уровней успешности построения tunnel, а также другие улучшения производительности.

Этот релиз обратно совместим, и если у вас уже установлен 0.6.1.31, подпись на подписанном файле обновления будет распознана (так что вы сможете обновиться автоматически).

Однако, чтобы обеспечить более равномерное распределение нагрузки между нашими тремя подписанными серверами обновлений, рекомендую выполнить следующую необязательную проверку:

* Open the update configuration page on:

http://localhost:7657/configupdate.jsp

* If you see 3 update URLs, no action is needed.

Если вы видите один URL-адрес обновления, откройте страницу расширенной конфигурации по адресу http://localhost:7657/configadvanced.jsp и удалите запись конфигурации, начинающуюся с "router.updateURL=". Затем нажмите "Apply", и для URL-адресов обновления должны вступить в силу настройки по умолчанию.

* Please note that since 0.6.1.31, configuration entries

может занимать несколько строк, поэтому, пожалуйста, перепроверьте, где начинается следующая запись, чтобы не повредить её!
