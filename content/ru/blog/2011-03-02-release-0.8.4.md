Релиз 0.8.4 содержит некоторые улучшения производительности и важные исправления ошибок. Кроме того, i2psnark теперь поддерживает магнет-ссылки. Как обычно, рекомендуется обновиться.

Пожалуйста, помогите развивать сеть. [Присоединяйтесь](http://www.i2p2.de/getinvolved.html), расскажите другим и [пожертвуйте](http://www.i2p2.de/donate.html)! Если вы обнаружите ошибку, пожалуйста, создайте отчёт об ошибке на [gitlab](http://i2pgit.org/i2p-hackers/i2p.i2p/issues). Мы по‑прежнему ищем помощь с новыми и существующими переводами. Пожалуйста, откликнитесь в IRC‑канале #i2p-dev.

Файлы доступны на [странице загрузки](/downloads/).

## Сведения о выпуске:

- Fix a severe memory leak in router I2CP session management that caused router crashes for people running the Robert bittorrent client
- Fix a bug from 0.8.2 that filtered cookies in the HTTP Server tunnel, causing authentication problems for some eepsites (I2P Sites)
- Several fixes for rare NPEs

## I2PSnark

- Magnet and "maggot" link support
- Peer Exchange
- Metadata transfer
- Details page enhancements
- Bandwidth limiter improvements
- Bug fixes

## Прочее

- More performance improvements
- Adjustment to peer selection, to hopefully use more of the available network capacity
- Translation updates
- Start of a new Arabic translation by 'hamada'
