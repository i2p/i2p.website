## Подробности обновления

0.9.29 содержит исправления для множества тикетов Trac, включая обходные решения для повреждённых сжатых сообщений. Теперь мы поддерживаем NTP по IPv6. Мы добавили предварительную поддержку Docker. Теперь у нас есть переведённые man-страницы. Теперь мы пропускаем заголовки Referer same-origin (одного источника) через HTTP-прокси. Есть дополнительные исправления для Java 9, хотя мы пока не рекомендуем Java 9 для общего использования.

Как обычно, мы рекомендуем обновиться до этого релиза. Лучший способ поддерживать безопасность и помогать сети — использовать последний релиз.
