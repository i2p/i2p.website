I2P Easy-Install Bundle for Mac (упрощённый пакет установки для Mac) уже в течение последних двух релизов не получает обновлений из‑за ухода его мейнтейнера. Рекомендуется пользователям Easy-Install bundle для Mac перейти на классический установщик в стиле Java, который недавно был возвращён на страницу загрузок. У версии 1.9.0 есть известные проблемы безопасности и она не подходит для размещения сервисов или какого‑либо долгосрочного использования. Пользователям рекомендуется как можно скорее мигрировать. Продвинутые пользователи Easy-Install bundle могут обойти эту ситуацию, собрав пакет из исходного кода и самостоятельно подписав программное обеспечение.

## Процесс нотаризации для MacOS

В процессе распространения приложения среди пользователей Apple есть много этапов. Чтобы безопасно распространять приложение в виде .dmg, оно должно пройти процедуру notarization (проверки Apple). Чтобы отправить приложение на notarization, разработчик должен подписать приложение с помощью набора сертификатов: один — для подписи кода и один — для подписи самого приложения. Это подписание должно выполняться на определённых этапах процесса сборки, до того как можно будет создать финальный .dmg-пакет, который распространяется конечным пользователям.

I2P Java — сложное приложение, и из‑за этого подбор соответствия между типами кода, используемого в приложении, и сертификатами Apple, а также определение того, где и на каком этапе выполняется подпись для получения действительной метки времени, происходит методом проб и ошибок. Из‑за этой сложности существующая документация для разработчиков не помогает команде понять правильную комбинацию факторов, которая приведёт к успешной нотаризации.

Эти сложности затрудняют прогнозирование сроков завершения этого процесса. Мы не узнаем, что всё сделано, пока не приведём в порядок среду сборки и не пройдём весь процесс от начала до конца. Хорошая новость в том, что количество ошибок в процессе нотаризации сократилось с более чем 50 при первой попытке до всего 4, и можно разумно предположить, что работа будет завершена до или как раз к следующему релизу в апреле.

## Варианты для новых установок и обновлений I2P на macOS

Новые участники I2P по-прежнему могут скачать Easy Installer для программного обеспечения версии 1.9.0 для macOS. Надеюсь подготовить релиз ближе к концу апреля. Обновления до последней версии станут доступны, как только нотаризация пройдёт успешно.

Классический вариант установки также доступен. Для этого потребуется загрузить Java и программное обеспечение I2P с помощью установщика на основе .jar.

[Инструкции по установке Jar доступны здесь](https://geti2p.net/en/download/macos)

Пользователи Easy-Install могут обновиться до той последней версии, используя локально собранную разработческую сборку.

[Инструкции по сборке Easy-Install доступны здесь](https://i2pgit.org/i2p-hackers/i2p-jpackage-mac/-/blob/master/BUILD.md)

Также можно удалить программное обеспечение, удалить каталог конфигурации I2P и переустановить I2P с помощью установщика .jar.
