Welcome to the I2P documentation. This site provides guides, explanations, and technical specifications for developers and users.

## Documentation Structure

- **[How I2P Works](/docs/how/)** - Conceptual explanations of I2P's design and operation
- **[Applications](/docs/applications/)** - Guides for using specific applications over I2P
- **[Guides](/docs/guides/)** - User and developer tutorials
- **[Development](/docs/develop/)** - Resources for I2P developers
- **[Reference](/docs/reference/)** - Quick reference materials (ports, etc.)
- **[History](/docs/history/)** - Historical design discussions

## Specifications

For detailed protocol specifications, see the [Specifications](/docs/specs/) section, which includes:

- **Transport Layer** - NTCP2 and SSU2 protocols
- **Tunnel Layer** - Tunnel building and routing
- **Core Protocols** - I2NP and I2CP
- **End-to-End** - Streaming and Datagrams
- **APIs** - SAM, BOB, I2PTunnel, and other client interfaces
- **Cryptography** - Encryption and signing algorithms
- **Data Formats** - Configuration and file formats

If you find any inaccuracies in the documentation, please [report an issue](https://i2pgit.org/I2P_Developers/i2p.www/issues).
